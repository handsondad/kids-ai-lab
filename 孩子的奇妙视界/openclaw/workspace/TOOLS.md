# TOOLS.md - 本地工作区说明

这里记录的是“孩子的奇妙视界”这套 OpenClaw 工作区的本地结构和使用约定。

## 关键路径

- 工作区根目录：当前目录
- 配置入口：`../openclaw.json`
- 配置包说明：`../README.md`
- 接入说明：`../OPENCLAW_INTEGRATION.md`
- 年龄框架：`AGE_STAGES.md`
- Prompt 框架：`PROMPT_PLAYBOOK.md`
- 能力说明：`SKILLS.md`

## 内容目录

- `notes/`：高价值主题笔记，优先于临时聊天回答
- `_templates/`：沉淀内容时使用的模板，不作为答案来源

## 写回约定

- 高频问题、成熟回答和可复用话题，优先写入 `notes/`
- 一个文件只保留一个核心问题
- 文件命名推荐使用 `topic-0001-short-slug.md`
- 回答脚本推荐使用 `script-0001-short-slug.md`

## 检索与生成约定

- 先读年龄框架，再决定怎么解释
- 先读 Prompt 框架，再决定回答结构
- 需要新鲜事实时再走网页检索
- 如果用户想把一个回答沉淀成长期资产，优先使用模板补成结构化笔记

## 多模态约定

当前默认环境更适合输出这些“多模态就绪内容”：

- 配图说明
- 旁白脚本
- 对话分镜
- 父子共做的小实验或观察动作

如果以后接入支持视觉或音频的模型，可以在保留现有文件结构的前提下继续升级。
