# 练习简介

这是一个适合 8-12 个月宝宝的共同凝视练习。核心不是让宝宝一直看着你，而是先跟他一起看同一个东西，再用指一指、递一递、说一说的方式，把注意、理解和词语连起来。这个阶段开始出现的共同凝视，是后续词汇和理解能力的重要基础。

## 适用场景

- 宝宝正在看玩具、宠物、窗外车辆或走进房间的人时
- 共读时指着图片一起看时
- 宝宝拿起或碰到某个东西时
- 白天散步、在家观察环境时

## 准备材料

- 一个宝宝已经在关注的真实东西
- 可以是玩具、猫、杯子、车、图书图片或家里某个人

## 练习步骤

1. 先跟着宝宝的视线或手去看他已经在关注的东西，不急着换主题。
2. 用手指靠近目标，帮助宝宝更容易把你的词和那个对象连起来。
3. 说一句很短的命名或描述，比如名称、动作、位置或一个简单属性。
4. 停一下，看宝宝会不会再看、再摸、再发声，或者去看你指的方向。
5. 熟悉后，可以加入一个很简单的请求，比如“把球给奶奶”或“看看小猫”，但一次只做一步。

## 家长可直接说的话

- “你在看猫，小猫在走。”
- “球在这里，你碰到球了。”
- “爷爷进来了，爷爷在这里。”
- “把球给妈妈，好，我们一起拿。”

## 为什么这样说有效

- 共同凝视让宝宝第一次把“我在看什么”和“你在说什么”稳定地连起来
- 先跟随宝宝注意点，再命名，比成人突然换主题更容易带出理解
- 真实物体、真实人和真实动作，比抽象讲解更容易帮助这个阶段的宝宝学词

## 观察点

- 宝宝会不会从你的手指转向目标，而不只盯着手指本身
- 他更容易对名字、动作还是位置词做出反应
- 简单请求时，宝宝是否开始表现出“听懂一点”的迹象

## 难度变体

- 简化版：只做命名和描述，不加请求
- 进阶版：在宝宝稳定后，再加入一个非常简单的动作任务，比如“给我”或“看这里”

## 安全或边界提醒

- 不要把共同凝视变成持续测试，重点是一起看，不是一直考
- 一次只盯一个对象，不要连续指多个东西让宝宝疲劳

## 检索提示

- 8-12 个月，共同凝视，指一指，说一说，描述宝宝关注的事物。
