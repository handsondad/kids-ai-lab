# 核心观点

- 阅读之所以对婴幼儿重要，不只是因为“多听了几个词”，还因为它把注意力、亲子联结和交流三件事同时拉到一起。孩子坐在大人怀里听、看、指、问，本身就是大脑发育的复合练习。
- 共同视觉关注是共读里非常关键的一环。孩子和大人一起看同一个图、同一个物件时，词语更容易真正和那个对象连起来；缺少这种共看的状态，词汇学习效果就会弱很多。
- 所谓“对话式阅读”不是把书读成课堂，而是在阅读过程中让孩子参与：描述图、数一数、猜接下来会怎样、联系自己的经验、说说角色感受。这比单向念完整页更容易把孩子拉进来。
- 早期读写前能力并不等于提前认字。更早也更重要的是：知道文字和图画不一样、文字有意义、文字有社会功能、书有基本的阅读惯例，以及现实中的标识和印刷符号代表真实事物。
- 18 个月以后，很多孩子开始注意生活里的文字和标识，例如包装、招牌、菜单、道路标记和自己的名字标签。这类“环境文字”能帮助孩子练习符号代替实物的思维，但不需要被立刻升级成正式教学。
- 很小的孩子摸书、翻错方向、咬书、只盯图不听故事，都是正常的读前行为。成人更需要做的是提供耐用、够得着的书，营造书随手可得的环境，并让孩子看到大人自己也在读。

## 家长会怎么问

- 读书给孩子听除了词汇多一点，还有什么用
- 共读时应该一直念文字，还是跟孩子聊图片
- 什么叫对话式阅读，怎么做才不变成考试
- 孩子乱翻书、咬书、只看图不听，是不是不适合读书
- 2 岁孩子开始认商标和招牌，这算阅读吗
