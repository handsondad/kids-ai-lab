# 核心观点

- 给孩子读书听不是提前教识字，而是在大脑发育关键期为语言、听觉处理和后续阅读打基础。越早开始越好，不需要等到孩子会说话，更不需要等到上幼儿园。
- 对很小的孩子来说，阅读最先提供的是声音、节奏、押韵和亲密互动。4 个月左右就可以开始把孩子抱在腿上读；更早时即使孩子只是听声音，也已经在接收有价值的语言输入。
- 有节奏、有押韵、有重复的文本特别适合婴幼儿，因为它们会反复强调语音模式和变化，这正是大脑后来分辨词音、建立听觉基础的重要材料。
- 共读能显著扩大孩子接触到的词汇和概念范围。书里会出现日常生活里不一定反复遇到的词和事物，这会帮助孩子更快建立更丰富的概念网络。
- 磁带、录音或单独播放朗读不能替代成人亲自读书给孩子听。阅读真正的力量不只是声音本身，而是声音、表情、注意力和亲密互动一起发生。
- 对 0-3 岁来说，更有效的做法通常不是一次读很久，而是把短时朗读变成每天反复出现的小 routine，让书和舒服、被陪伴的时刻稳定地绑定起来。

## 可拆出的阶段卡或练习卡

- 0-12 个月越早开始越好的朗读和听觉输入阶段说明
- 0-6 个月节奏、押韵优先的早期共读练习
- 18-30 个月利用重复文本和熟书做轻量填词的共读练习
