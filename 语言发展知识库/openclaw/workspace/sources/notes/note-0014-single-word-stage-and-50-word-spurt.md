# 核心观点

- 单个词阶段常会持续一段时间，而且前 50 个稳定词往往增长得很慢；但当孩子接近或达到大约 50 个词时，常会出现明显的词汇加速，逐步迈向双词组合。
- 早期稳定词通常来自高频、可见、可操作的生活类别，例如家人称呼、动物、食物、衣物、交通工具、家居用品、玩具、问候语和少量高频形容词。
- 研究提示，很多 1 岁半左右的孩子平均每天还能从生活里继续学到 2-5 个词，这使得“顺着日常持续描述”比“专门教词或考词”更关键。
- 这个阶段发音上的“错误”变多，往往并不意味着退步，而是孩子正在更主动地练习词音、尝试更精细地区分和组织语音。
- 有些看起来像两个词连在一起的表达，其实更像被整体记住的“大词”，并不等于已经稳定掌握了真正可拆开的双词结构。
- 也有些孩子会先用带停顿的一串单词表达意思，再慢慢把这些单词连得更紧，过渡到真正的短句。

## 家长会怎么问

- 孩子大概会说多少词时开始突然说很多
- 首词阶段最常先学会哪类词
- 为什么孩子最近说得更不清楚了
- 孩子把两个词粘在一起说，算不算会说短句了
- 每天到底该怎么帮孩子慢慢扩词汇
