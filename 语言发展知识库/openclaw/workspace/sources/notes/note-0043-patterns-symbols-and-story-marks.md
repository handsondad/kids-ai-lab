# 核心观点

- 本章把很多看起来“不是语言课”的活动，统一放回读写前能力的底层：配对、找规律、识别标识、假装对话、涂鸦讲故事，都是在帮孩子练“声音、图形和符号都能代表某种意义”。
- 6-18 个月时，孩子可以先从颜色配对、简单图书对话和熟悉包装或标识的整体辨认开始。重点不是立刻说出颜色名或认出字母，而是先学会看出一样和不一样、同类和不同类，以及“这个符号代表那个东西”。
- 用真实包装、餐饮袋、空盒子和熟悉标识玩过家家，是很早的环境文字练习；再往后把少量标识做成卡片，让孩子给实物配对，会进一步稳住“标识代表实体或概念”的理解。
- 玩具电话之所以好用，不只是因为好玩，还因为道具会降低说话压力。孩子更容易模仿打招呼、问名字、轮流说一句的对话框架，并把它带回真实交流。
- 找规律、串珠、触觉多米诺和同类配对，会同时练视觉或触觉辨别、分类整理和模式预测。这些能力对之后的阅读、书写、数学入门都很关键。
- 粉笔画、涂鸦和“你说我写”很重要，因为它们让孩子体验到：纸上的痕迹是有意义的；我说的话可以被写下来；图片、故事和文字之间是可以互相转换的。

## 可拆出的阶段卡或练习卡

- 12-36 个月从配对、模式到符号理解的阶段说明
- 把玩具电话变成低压力来回对话的假装交流练习
- 把涂鸦、讲述和成人代笔连起来的“小故事书”练习
- 用真实包装和少量标识卡做环境文字配对的家庭练习

## 家长会怎么问

- 孩子总爱认商标、包装和路牌，这算阅读吗
- 为什么先会配对颜色和图样，比先会说颜色名字还常见
- 玩具电话为什么会让孩子更愿意说话
- 乱涂乱画真的和以后写字有关系吗
- 要不要现在就拿卡片正式教认字母
