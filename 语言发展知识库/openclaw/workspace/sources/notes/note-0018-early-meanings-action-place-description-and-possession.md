# 核心观点

- 在 9 个月到 2 岁这段时间里，孩子开始表达的意义迅速增加；他们说出的词也许和成人词形不一样，但依然能非常清楚地完成沟通。
- 早期单词并不只是在给物体命名，也常被孩子拿来表示动作、位置、描述、占有和拒绝等不同含义。
- 孩子会用自己的方式把物体名称或声音当成动作来表达，例如用车的声音来表达“车在走”，这类表达先抓住了意义，再逐渐过渡到更常规的动词。
- 位置类含义也很早出现，例如“那”一开始常常不只是方位，还带有“我做完了”“它在那里”的功能。
- 描述性词语常会很早进入表达，尤其是高频、对孩子有现实意义的词，例如“烫”“美”“大”。
- 占有也是重要的早期意义之一，尤其在和兄弟姐妹或同伴互动时，“我的”会很早变得有用。
- 成人是否以对孩子有意义的方式回应，会直接影响孩子能不能把这些早期意义稳稳建立起来；面对面互动仍然是最强的学习场景。

## 可拆出的阶段卡或练习卡

- 早期单词不只是在命名的阶段说明
- 看情境理解动作词、位置词和占有表达的回应练习
- 重视孩子自创标签和早期描述词的日常接话练习

## 家长会怎么问

- 宝宝一直说车的声音，是不是不算会说词
- 孩子老说“那”，到底是在说哪里还是说做完了
- 为什么“烫”“漂亮”“大”这类词会这么早出现
- 宝宝总说“我的”，是不是太自私了
- 孩子自己给东西起名字，要不要纠正
