# 核心观点

- 孩子通常不会一下子把所有特殊疑问词都用好。what 和 where 往往更早、更容易稳定；who 常是下一步；when 和 how 对很多孩子来说要更晚才能真正听懂；why 往往是最容易把大人问崩溃的那个。
- 大人如果想帮孩子练提问，最值得优先给的是更具体、负担更低的问题，比如“谁在做这个”“小狗去哪里了”“这是什么”，而不是太早大量追问 how 和 why。
- 重复阅读喜欢的书，是练提问的高价值场景，因为孩子已经熟悉内容，更有余力玩问答形式本身。
- 翻翻页、躲藏角色、忙碌的细节图和滑稽插图书，都很适合支持“谁”“哪里”“接下来呢”这些问题；随着年龄增大，才慢慢把 why 和 if...then 类问题放进去。
- 当家长已经很疲惫时，可以适当限制提问游戏的时长。保持可持续的互动，比勉强把每一轮都答到极致更重要。

## 可拆出的阶段卡或练习卡

- 疑问词从 what/where 到 why/if...then 的阶段说明
- 重复共读中练“谁/哪里/接下来”的家庭练习
- 用固定收尾语结束提问游戏的家长策略卡

## 家长会怎么问

- 先教什么问题更合适
- 为什么孩子总问 why，却听不懂答案里的时间信息
- 什么书最适合带问题一起读
- 我能不能限制提问时间，不然真的撑不住
