# 核心观点

- 语言发展前面有很多看起来不像“语言”的铺垫，包括把一个物体假装成另一个物体、会使用简单工具、能记住别人做过的事并随后模仿。
- 这些认知和游戏能力与语言关系很深，因为它们都涉及“一个东西可以代表另一个东西”“先看到、记住，再复现”的能力，而这和词语代表事物的方式是相通的。
- 1 岁半到 2 岁左右的孩子经常通过感官和动作学习，也会热衷重复某个动作，例如清空箱子再装回去，这既是游戏，也是学习。
- 对象永久性、工具使用、模仿记忆和假装游戏，都是语言能够进一步发展的重要地基。
- 孩子最开始并不会真正“和别人一起玩”，常先经历同处一室玩、并排玩、互相模仿但不怎么对话的阶段；真正共享玩具、一起推进游戏和聊天，往往要更晚一些，3 岁仍觉得所有玩具都是自己的也不算罕见。
- 因此，语言支持不只发生在“说话练习”里，也发生在孩子玩、模仿、假装和逐渐学会与别人一起玩的过程中。

## 可拆出的阶段卡或练习卡

- 18-24 个月词汇增长阶段中的假装游戏与重复动作
- 3-4 岁角色扮演与一起玩耍的语言支持
- 假装一个物体是另一个物体时的描述式陪玩
