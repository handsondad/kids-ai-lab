# 核心观点

- 家长说得多不一定更有效，更关键的是说得准、说得贴着孩子当下的注意点，并且留下轮到孩子回应的空间。
- 对年龄更小的孩子，命名、模仿、等待、跟随注意点，比连续提问更有效。
- 对刚起步表达的孩子，“扩展”比“纠错”更有效。孩子说得短，成人只往上加一步，不要一口气加成长句。
- 对更大的孩子，开放式提问、复述、重组和轻度 recast 更能支持叙事和社交语言，而不是不断追着问“对不对”。
- 真实日常场景是最稳定的语言训练场。吃饭、换衣、看书、散步、睡前复盘，往往比专门安排的“训练时间”更可持续。

## 可拆出的阶段卡或练习卡

- 面对面轮流发声
- 二选一加等待回应
- 单词加一步的扩展说法
- 场景描述加一个跟进问题
- 角色扮演对话梯子
- 睡前回顾小故事

## 家长会怎么问

- 我该多说一点还是少说一点
- 孩子只说一个词，我下一句应该怎么接
- 怎么说才不是在考孩子
- 睡前怎么聊天既安静又能练表达
- 排队等候时怎么聊更有价值
