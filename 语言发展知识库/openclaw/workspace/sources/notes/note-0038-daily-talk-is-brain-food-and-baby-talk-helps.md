# 核心观点

- 促进孩子大脑和语言发展的“便宜好药”不是额外购买的产品，而是日常对话本身。和孩子说话这件事越早开始越好，从出生就可以做，而且在 0-3 岁会持续累积效果。
- 对孩子来说，重要的不只是有没有听到语言，还包括听到的话够不够多、语气是不是更积极、更温和、有没有真实的来回互动。书中引用的研究强调，这些差异和后续智力、词汇与理解表现有关。
- 说什么语言并不是核心障碍。家庭里用自己最自然、最熟悉的语言和孩子交流，同样有价值；为了“等以后学英语”而刻意少说，反而会减少孩子早期获得的语言输入。
- 所谓“宝妈腔”并不是多余夸张，而是婴儿更容易处理的一种输入方式：更慢一点、更高一点、更有重音一点、元音拉长一点。这种说话方式会自然地帮助孩子抓住语调和词边界。
- 给孩子念书、边做边说、唱歌、在不同房间里抓住可聊的对象，都属于有效输入。录音、磁带或课程本身不能替代日常的人际交流；真正有力量的是贴着孩子、贴着场景的真实说话。

## 可拆出的阶段卡或练习卡

- 0-12 个月“从出生就开始听和学”的阶段说明
- 12-24 个月把日常类别词藏进 routine 的扩词练习
- 护理、吃饭、出门和洗澡时的解说式对话练习

## 家长会怎么问

- 宝宝刚出生，我需要等他会回应了再开始说话吗
- 我不知道该说什么，随便聊家常也有用吗
- 我的母语不是英语，或者保姆英语不好，会不会耽误孩子
- 给孩子听磁带、上外语班，能不能代替日常聊天
- 宝妈腔是不是太夸张了，会不会没必要
