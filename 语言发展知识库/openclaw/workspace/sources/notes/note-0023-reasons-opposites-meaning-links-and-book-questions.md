# 核心观点

- 当孩子能组合更多词语后，也会开始接触更复杂的含义，例如理由、反义词，以及不同事物之间基于外形、经验或功能的联想关系。
- 小孩子很早就会给出理由，但常常不是成人式的“因为……”句子；他们可能只说“洗澡”来表达“我不想吃了，我想去洗澡睡觉”。
- 反义词通常不是成对同时学会的，常常先学会一个更常见的词，例如“烫”早于“冷”、“大”早于“小”，所以他们会用已知词去覆盖一整类感觉或程度。
- 孩子会基于自己经验建立含义之间的联系，因此可能把吊灯叫“鸭子”、把盘子叫“月亮”；这些出乎意料的命名，往往透露的是他们已经在主动做概念关联。
- 研究提示，学习后的小睡有助于保留新学到的东西，这也让睡前阅读和学习后短暂休息更有价值。
- 阅读是帮助孩子理解这些复杂含义的高价值场景：简单数数书、能问“接下来呢”“在哪里”“为什么”的书，以及与孩子自己生活经验相关的故事，都很有帮助。
- 对更复杂的含义，问问题是有价值的，但问题难度要循序渐进；从具体的“在哪里”“接下来呢”到更难的“为什么”，中间通常要经历很长一段时间。

- 孩子为什么会把完全不同的东西叫成同一个名字
- 为什么孩子会说冰激凌“烫”
- 孩子什么时候才会真正说出“因为……”
- 看书时到底该问什么问题
- 数数书和细节很多的图画书分别有什么用
