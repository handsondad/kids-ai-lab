# 核心观点

- 对接近入学年龄的孩子来说，持续聊天仍然非常重要。孩子随时会说话、会对路上和家里几乎所有东西感兴趣，而这些随手聊出来的内容，本身就在喂养语言系统。
- “回忆今天发生了什么”是很有价值的语言练习。加餐、洗澡、睡前这些固定时段，尤其适合把一天里的一件小事重新说一遍；成人不需要讲太多，只要发起话题，再给一两句评论就够了。
- 重复阅读熟书对语言发展很重要。孩子会想一遍又一遍地听同一本书，也会开始想把故事“读”给大人听。这种复述不一定是真的识字，但它在帮助孩子建立书本结构、故事顺序和阅读习惯。
- 有些孩子会开始对字母、数字和自己的名字感兴趣，但这不需要被提前变成正式教学。重点仍是兴趣、熟悉度和亲子共读，而不是过早追求独立认字。
- 不是所有孩子都爱坐着聊天或读书。有些孩子更爱做事、跑动和操作，但这并不意味着不需要书。对这类孩子，睡前或过渡时段的短时共读往往更可持续。

## 可拆出的阶段卡或练习卡

- 3-6 岁日常回忆和事件复盘的练习卡
- 让孩子把熟悉的书“读给你听”的共读练习
- 对字母、数字和自己名字产生兴趣时的轻量支持卡

## 家长会怎么问

- 幼儿园回来后到底该怎么聊，孩子才会多说一点
- 孩子总要重复同一本书，要不要制止
- 他说是在“读书给我听”，但其实不是逐字读，这样有价值吗
- 孩子只爱跑来跑去，不爱坐着看书怎么办
