# 核心观点

- 孩子即使只会说单个词，甚至还没有多少“正常词语”，也已经在用沟通完成很多不同的事情，而不只是给物体命名。
- 早期单词常常承担不同功能：想要东西、让别人做事、邀请一起玩、表达当下兴趣、探索“那是什么”、自言自语练习和想象、偶尔报告一件刚发生的事。
- 成人是否看见孩子正在看什么、拿什么、做什么，往往决定了能不能理解这些单词真正的意思；同一个近似发音在不同情境里可能指向完全不同的东西。
- 单个词阶段并不意味着孩子只能“说一个名称”，他们已经在用语言影响别人、组织互动、发起游戏和表达自己的偏好。
- 等孩子把不同的词和不同的语言功能开始放进同一个短句里，沟通就会进一步复杂起来，这也为后续更完整句子打基础。

## 家长会怎么问

- 孩子只会说单个词，到底算不算在“真正说话”
- 为什么同一个词在不同场景里意思不一样
- 宝宝老说“那个”是在问问题吗
- 为什么孩子还不会句子，却已经能指挥大人了
- 孩子跑来跟我说一个词，是想要东西还是在告诉我事情
