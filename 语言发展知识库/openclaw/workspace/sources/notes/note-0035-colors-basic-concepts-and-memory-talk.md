# 核心观点

- 颜色对成人来说很自然，但对孩子来说并不简单。孩子要先慢慢理解“相同”和“不同”，才更可能真的掌握颜色；很多时候他们会先会配对颜色，再会说出颜色名称。
- 如果要看孩子是不是真的在分颜色，最好用除了颜色外其他条件都一样的物品。否则孩子可能其实是在看大小、形状、材质或图案，而不是在看颜色本身。
- 除了颜色，入学前还会持续学很多基础概念词：大和小、前和后、上和下、开和关、一样和不一样、热和冷、轻和重等等。成人越能在真实场景里反复说这些词，孩子越容易理解并学会分类。
- 聊孩子自己的经历，会帮助他们建立记忆。活动前说说要发生什么，活动后再回忆看见了什么、做了什么、谁来了、哪一个更多或更大，这些都像在帮孩子慢慢写“自己的日记”。
- 不需要特地买专门教颜色和数字的书。很多普通图画书、熟书、照片和外出场景里，本来就有大量数量、颜色、形状、位置和比较机会；关键是成人怎么聊，而不是材料多特别。

## 可拆出的阶段卡或练习卡

- 3-4 岁颜色匹配先于颜色命名的阶段提醒
- 入学前基础概念词和分类词持续扩展的阶段说明
- 在游戏、指令、儿歌、外出和共读里反复说基础概念的家庭练习
- 通过聊经历帮助孩子形成“自己的日记”的回忆练习

## 家长会怎么问

- 为什么孩子会挑出不一样的颜色，却还叫不出颜色名字
- 蓝色和绿色为什么总是分不清
- 基础概念词到底要怎么教，才能不是上课
- 3 岁多以后，怎么聊孩子的经历才更能帮他记住发生过的事
