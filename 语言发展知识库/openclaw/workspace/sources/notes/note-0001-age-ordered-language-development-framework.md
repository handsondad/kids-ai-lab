# 核心观点

- 语言发展不是从“开口说话”才开始，而是从共同注意、轮流互动、声音回应和手势沟通就已经开始了。
- 理解通常早于表达。很多孩子会先表现出“听懂更多”，再慢慢把这些理解转成单词、短语和句子。
- 0-6 岁的语言支持最好按年龄阶段看重点，而不是用同一种说话方式覆盖所有年龄。
- 年龄越小，越需要在日常 routine 里反复、短句、紧贴当下地说；年龄越大，越需要增加开放式提问、回顾、推理和叙事。
- 家长最有效的支持通常不是反复考问孩子，而是跟随孩子注意点、把语言嵌进真实互动，并留下可回应的空档。

## 可拆出的阶段卡或练习卡

- 0-12 个月：前语言期的联结、轮流和声音回应
- 12-18 个月：理解增加、手势和单词起步
- 18-24 个月：词汇增长和双词组合
- 24-36 个月：简单句和往返对话
- 3-4 岁：提问增多、角色扮演和简单叙事
- 4-6 岁：叙事组织、社交语言和早期读写前能力

## 家长会怎么问

- 0-1 岁要怎么和宝宝说话才有帮助
- 1 岁半只会说几个词，要重点做什么
- 2 岁孩子怎么从单词过渡到短句
- 3 岁怎么在日常生活里练对话
- 4-6 岁怎么练复述和表达更完整
