# 核心观点

- 一旦宝宝开始说出几个词，大人最有价值的做法之一是重复他们说过的话，并把这个词放进非常短的句子里，帮助他们把单词和更完整表达联系起来。
- 在这个阶段，句子要短，最好只包含一到两个关键信息点，而且可以把重点词放在句首或句尾，减少中间信息过多带来的理解负担。
- 不需要“教导”式提问或复杂讲解，而是持续描述正在发生的事、描述宝宝正在做的动作，尤其是在重复性活动里不断示范高频词，会很有帮助。
- 宝宝在这个阶段常会反复做同一个动作，例如把东西拿出来、放进去、清空盒子、再装回去。这样的重复活动天然适合练“出来”“进去”“没有了”“还有”这类高频词。
- 描述要贴着宝宝正在看的和正在做的东西，而不是讲宝宝没有在关注的对象。颜色这类难度较高的信息，在首词阶段并不需要优先强调。
- 配有熟悉物体的大图书很适合这个阶段。一起指认宝宝已经会说或快要会说的词，并在图片里“找东西”，会帮助他们把词语和图片、再和真实世界连接起来。

## 可拆出的阶段卡或练习卡

- 重复孩子词语并加一句短句的练习
- 重复性活动中的“出来/进去/没有了”词语练习
- 图片里找熟悉词和指认图片的共读练习

## 家长会怎么问

- 宝宝说“抱”，我下一句该怎么接
- 1 岁多要不要说完整句子给他听
- 宝宝总是重复拿出来放进去，我该怎么陪他说话
- 这个阶段读图画书应该怎么读
- 为什么不要一直强调颜色
