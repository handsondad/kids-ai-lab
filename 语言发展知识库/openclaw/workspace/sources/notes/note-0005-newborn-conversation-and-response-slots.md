# 核心观点

- 和新生儿说话时，说什么并不是最关键的，关键是持续地和他们说话，让他们在真实互动中听到母语、注意人脸、体验轮流。
- 家长可以谈任何话题，包括天气、自己接下来要做什么、正在做什么，以及刚刚做完什么。这样的日常解说，本身就是语言输入。
- 和小婴儿“聊天”时，最重要的技巧之一是留出回应时间。先说一句，再停几秒，等宝宝用声音、表情或动作回应。
- 宝宝的回应不一定是发出声音。挥手、踢腿、打嗝、做表情、看向你，都可以看作他们在用自己的方式接话。
- 眼神交流、抬眉、睁大眼睛和夸张表情，能够帮助婴儿理解现在轮到谁“说话”。
- 换尿布、洗澡、坐宝宝椅这些高频 routine，不只是护理时刻，也是最适合建立早期对话节奏的时刻。
