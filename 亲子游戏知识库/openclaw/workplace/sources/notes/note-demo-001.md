# 核心观点

- 2-3 岁孩子适合通过简短口头指令进行语言理解练习。
- 游戏越短越容易保持参与度，最好在 5-10 分钟内完成一轮。
- 家长的命名、重复和等待回应，比一次讲很多更有效。

## 可落地玩法

- 听声找物：让孩子根据声音或名称找到对应物品。
- 二选一指令：让孩子在两个选项中做出回应。
- 房间寻宝：把语言提示和空间位置结合起来。
