# 专题说明

这个合集适合在孩子容易紧张、过渡卡顿、睡前拉扯或家庭节奏不稳时使用。它把“先用身体连接稳下来”“先看孩子的频道”“先安抚再推进”“用熟悉小物做桥梁”“用固定摇篮曲慢慢降下来”“把睡前做成固定流程”“用固定告别脚本建立可预期感”“用小仪式建立稳定感”放到一组，现在也补进了膝盖摇篮曲、抱着慢舞、飞机抱、毯子轻摇、路上家常聊天和洗澡话语歌这些更低月龄到婴儿期都能直接使用的过渡入口。

## 推荐顺序

1. 低月龄或刚受惊时，先用 `game-0021`、`game-0022` 或 `game-0023` 做身体上的安稳连接。
2. 进入具体过渡前，再用 `game-0016` 判断现在该快一点还是慢一点，不急着直接推任务。
3. 当孩子已经紧起来时，优先用 `game-0017`、`game-0063` 或 `game-0018` 把状态降下来。
4. 面对出门、坐车、探访或洗澡这类日常过渡时，可以用 `game-0044` 和 `game-0048` 先把节奏与预告变清楚。
5. 面对短暂分开、交接照护或出门时，可以固定用 `game-0034` 让告别和重逢变得可预期。
6. 晚上固定用 `game-0019` 收住一天，减少睡前来回拉扯。
7. 长期想建立家庭稳定感时，再加入 `game-0020` 作为每周或每月会回来的固定锚点。

## 包含游戏

- `game-0021`: 胸贴胸静静抱，适合低月龄先用体温和心跳帮助稳定下来
- `game-0022`: 背带安抚散步，适合在移动和过渡里保持贴身安定感
- `game-0023`: 婴儿按摩身体对话，适合把抚触和语言一起做成短时安抚活动
- `game-0016`: 频道对一对，适合在各种过渡前先做短时观察和匹配回应
- `game-0017`: 轻声慢唱安抚，适合孩子受惊、犯困或情绪上来时先降刺激
- `game-0063`: 摇一摇慢慢唱，适合用固定摇篮曲和轻摇把身体节奏慢慢放下来
- `game-0086`: 膝盖摇篮曲，适合用大腿小摇篮做更贴近新生儿的轻摇安抚
- `game-0088`: 抱着跳支慢舞，适合把贴身抱持、旋律和步伐合成一段温柔过渡
- `game-0094`: 飞机宝宝慢慢飞，适合在俯趴托抱和小幅摆动里安抚哭闹宝宝
- `game-0097`: 毯子摇摇小吊床，适合两位成人一起做低幅度、可预期的轻摇连接
- `game-0018`: 安心小物陪伴，适合把拥抱和入睡经验逐步转成可带着走的熟悉线索
- `game-0044`: 一路唠唠家常，适合在推车、坐车和换场景途中用短句提供稳定预告
- `game-0048`: 洗澡话语歌，适合把洗浴流程变成可预期的触觉加语言过渡
- `game-0034`: 早上好，再见，适合把短暂分开和重逢做成固定的小脚本
- `game-0019`: 睡前三步安心流程，适合把刷牙、讲故事和说晚安固定成稳定顺序
- `game-0020`: 家庭小仪式日，适合用重复出现的小传统建立归属感和可预期感

## 使用建议

- 一次只抓一个问题先解决，比如先处理“睡前乱”或“出门总崩”，不要一口气全改
- 如果孩子本来就很敏感，先把顺序稳定下来，再谈挑战和练习
- 家长自己的语气和节奏是这个合集里最关键的材料，比额外买什么都更重要
