# 专题说明

这个合集适合 1-2 岁孩子开始进入“我想自己动手”和“我想自己找”的阶段。活动重点是把注意力和动作连起来，让孩子在配对、舀倒、寻找这些任务里练习持续关注、眼手协调和简单问题解决。现在也加入了形状洞洞配对、冰箱贴找不同、倒瓶回装、管道滚球和循声找人的活动，把 18 个月前后更明显的操作和搜索欲也接了进来。

## 推荐顺序

1. 先用 `game-0011` 藏物捉迷藏热身，快速把孩子带进“找一找”的状态
2. 再用 `game-0009`、`game-0075` 或 `game-0199` 做图物、颜色或形状配对
3. 然后加入 `game-0188`、`game-0192`、`game-0200`、`game-0201` 或 `game-0202`，把开盒、缺失记忆、倒瓶和管道问题解决放进同一条探索线
4. 最后用 `game-0203` 或 `game-0010` 收尾，让循声找人或勺挖倾倒把专注从桌面延伸到你来我往的互动里

## 包含游戏

- `game-0009`: 图卡实物配对，适合把图片和现实物体连接起来
- `game-0010`: 勺挖倾倒，适合练习眼手协调、容器转移和短时专注
- `game-0011`: 藏物捉迷藏，适合让孩子在移动和寻找中练习注意转移
- `game-0075`: 彩环慢慢配一对，适合从颜色异同开始进入更清楚的配对活动
- `game-0188`: 我猜我猜打开盒子看看是谁，适合把开盒动作、熟悉图片和命名连起来
- `game-0192`: 罐中取宝拧一拧拿出来，适合练习透明目标、拧盖和取物操作
- `game-0199`: 形状搭配放回正确的洞洞里，适合把旋转、试放和成功嵌合做得更明确
- `game-0200`: 神奇冰箱贴挪一挪猜猜少了谁，适合把观察、记忆和“少了什么”组织成短任务
- `game-0201`: 麦圈挑战赛倒出来再放回去，适合把倒出目标和再装回去做成可重复操作
- `game-0202`: 管子里的秘密看球消失又回来，适合把隐藏路径、等待和终点回出现象连起来
- `game-0203`: 捉迷藏循声找人，适合把寻找动机和声音线索带进更主动的亲子搜索

## 使用建议

- 一次选 1-2 个就够，孩子很投入时再顺延到第三个
- 操作类活动尽量安排在孩子不困、不饿、愿意坐一会儿的时候
- 如果孩子明显烦躁，优先回到“找一找”或“藏物”这类更轻松的游戏

## 检索提示

- 1-2 岁，室内，动手操作，找一找，眼手协调，低成本，无屏幕。
