# 核心观点

- 这一章中很关键的一部分，是把“阅读前能力”写成孩子逐步理解符号系统的过程。孩子会先明白图画不是实物，随后慢慢明白文字也不是画，而是另外一种能代表事物、传达意思的符号。
- 很多孩子在 3 岁前后已经知道文字“有所指”，所以会问包装、菜单和招牌“上面说什么”。这说明他们已经在意识到文字是有意义、有用途的，而不只是背景装饰。
- 环境文字特别重要，因为它不是抽象练习，而是生活里真实出现的符号。商品包装、餐馆招牌、道路标识和熟悉的品牌图案，会帮助孩子把颜色、形状、图像和词语逐步连起来。
- 孩子辨识环境文字通常会经历从真实物体，到彩色标识，再到黑白标识，最后到标准字体词语的过程。大人可以从这个过程观察孩子的具象思维和视觉辨析到了哪一步，但不需要把它做成正式测试。
- 行文惯例也属于早期阅读经验的一部分。封面、页码、字母、单词、空格、从左往右、从上到下，这些规则不是靠讲解记住的，而是在一起看书、一起指字时逐渐熟悉的。

## 对家庭实践最有价值的线索

- 读书前能力并不只发生在书里，路上、超市、餐馆和家里包装盒上都有机会。
- 先让孩子在真实情境里认“这个标记代表什么”，再慢慢走向更抽象的文字辨识，更符合发展顺序。
- 点出“这是图画，那是字”，以及“我们从这边开始看”，能自然帮助孩子建立行文概念。
- 用环境文字做观察时，重点是看孩子怎么理解，不是追求答对率。

## 可落地玩法

- 图画文字页码看一看：在共读时顺手指出图、字、封面和页码。
- 出门认认招牌字：在真实环境里留意熟悉标识和包装文字。
- 从包装到词语的识字梯：用真实包装到简化文字的层级做观察和比较。
