# 游戏简介

这是一个适合 30-36 个月孩子的装饰和收纳准备活动。家长和孩子一起给纸箱、玩具箱或收纳盒贴贴纸、画标记、贴图片，让“玩具回家”这件事更有归属感。

## 适用场景

- 孩子已经有固定玩具，也愿意参与一点点收拾时
- 想把创作和日常秩序连接起来，而不是分开做时

## 准备材料

- 一个纸箱、收纳盒或玩具箱
- 贴纸、胶带、可水洗马克笔
- 可选：玩具图片或简单图标

## 玩法步骤

1. 先选一个最常用的盒子，告诉孩子这是玩具要回家的地方。
2. 一起决定要贴什么，比如车车、积木或颜色点。
3. 让孩子自己贴几张贴纸或画几笔标记。
4. 装饰完后，顺手放进 2-3 件对应玩具，建立“这个箱子装这些”的联系。
5. 之后几天继续使用同一个箱子，让标记慢慢有意义。

## 引导话术

- “这是车车的家。”
- “你想贴在上面还是前面？”
- “我们把玩具送回它的箱子里。”

## 观察点

- 孩子更投入装饰过程，还是更投入把玩具放回去
- 会不会开始主动根据标记找对应箱子
- 哪种标记最容易理解：颜色、图片，还是简单画线

## 难度变体

- 简化版：只做一个箱子的一个标记
- 进阶版：给两个箱子分别做不同标记，再试着分类放回

## 安全提示

- 盒子边缘要平整，避免纸板划手
- 小贴纸和胶带要在成人看护下使用，避免入口

## 检索提示

- 30-36 个月，装饰玩具箱，收纳标记，贴纸活动，参与收拾。
