# 游戏简介

这是一个适合 24-36 个月孩子的“翻包”游戏。家长给孩子准备一个属于她的小包，里面放几样安全的“大人物件”，比如梳子、旧钱包、镜子、钥匙模型或笔记本，让她翻一翻、摸一摸，再说说这些东西有什么用。

## 适用场景

- 孩子总想翻大人的包，对里面的东西特别着迷时
- 想把这种兴趣转成一个可控又有词汇价值的活动时

## 准备材料

- 一个安全的小包或小提包
- 4-6 样大件、安全、无尖角的小物件

## 玩法步骤

1. 先把所有物品都确认安全，再放进孩子的包里。
2. 让孩子自己把东西一件件拿出来，不急着提示太多。
3. 每拿出一件，就顺手说名字和用途。
4. 也可以让孩子只用手摸一摸，再猜里面是什么。
5. 熟悉后定期换几样物品，保持新鲜感。

## 引导话术

- “你摸到了什么呀？”
- “这是梳子，可以梳头发。”
- “这个钱包里可以放卡片。”

## 观察点

- 孩子更喜欢单纯翻找，还是更喜欢听你解释用途
- 会不会特别执着某一种“大人专属物件”
- 更愿意用手摸着猜，还是喜欢直接拿出来看

## 难度变体

- 简化版：只放 3 样最熟悉的物件
- 进阶版：让孩子根据你的描述从包里找出指定物品

## 安全提示

- 不放真实药物、钥匙、化妆品、尖笔或任何危险物
- 所有物件都要足够大，避免入口风险
