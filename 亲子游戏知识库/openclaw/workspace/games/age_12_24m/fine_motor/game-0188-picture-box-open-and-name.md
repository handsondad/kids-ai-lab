# 游戏简介

这是一个适合 12-24 个月孩子的开盒猜图游戏。家长在盒盖内侧贴上家人照片或熟悉物品图片，让宝宝打开盒子看一看、说一说，慢慢把“开盖动作”“看到的图片”和“熟悉名字”连起来。

## 适用场景

- 宝宝已经喜欢开盖、翻盖和重复看熟悉人物时
- 想把记忆和命名放进一个动手小游戏时

## 准备材料

- 2-3 个容易打开的小盒子
- 家人照片或熟悉物品图片

## 玩法步骤

1. 先只准备一个盒子，让宝宝打开看到里面的图片。
2. 你先示范一次，说出“这是爸爸”或“这是球”。
3. 再让宝宝自己打开，再听你说一次名字。
4. 熟悉后加第二个盒子，让他慢慢记住哪个盒子里是谁。
5. 每次只玩少量盒子，保持成功率。

## 引导话术

- “打开看看里面是谁。”
- “这是妈妈的照片。”
- “你找到球球啦。”

## 观察点

- 宝宝更在意开盒动作本身，还是更在意里面的图片
- 哪类图片最容易引起反应：家人、宠物还是熟悉物件
- 第二次玩时，会不会开始记得哪个盒子里有什么

## 难度变体

- 简化版：全程只玩一个盒子
- 进阶版：准备两个盒子，问“爸爸在哪个盒子里”

## 安全提示

- 盒子边缘要平整，不用容易夹手或松脱小件的盒子
- 图片固定牢，不让宝宝撕下入口

## 检索提示

- 12-24 个月，开盒游戏，照片记忆，视觉记忆，熟悉物命名。
