# 游戏简介

这是一个适合 12-24 个月孩子的经典童谣动作游戏。家长从几首短小、押韵、重复度高的儿歌里挑 1-2 首，一边唱，一边给每首歌配上固定手势和小动作，让宝宝在节奏和重复里慢慢熟悉语言的味道。

## 适用场景

- 想把一堆经典儿歌真正唱起来，而不是只停留在歌词上时
- 宝宝喜欢重复同一首歌，也喜欢等某个熟悉动作出现时

## 准备材料

- 不需要额外材料
- 1-2 首你最顺口的短童谣

## 玩法步骤

1. 先只选两首歌，每首配一个最明显的动作。
2. 每唱到关键词就做同一个动作，让结构变得可预期。
3. 一首歌唱两三遍，再换下一首，不要不停换新歌。
4. 如果宝宝开始期待某个动作，就故意停半拍，让他先笑一笑或做一点回应。
5. 用同一句结束语收尾，让他知道串烧结束了。

## 引导话术

- “这首歌唱到这里，我们来划船。”
- “小蜘蛛要爬上去啦。”
- “这次轮到小玛菲特坐下。”

## 观察点

- 宝宝更偏爱哪一首歌和哪一种动作
- 第二遍和第三遍时，是否开始提前期待动作出现
- 喜欢坐着听，还是会想站起来跟着动

## 难度变体

- 简化版：全程只唱一首歌
- 进阶版：两首歌之间加入“轮到下一首”的短过渡语

## 安全提示

- 所有动作都保持小而清楚，不做容易失衡的大幅晃动
- 宝宝已经分心时，不把串烧越拉越长

## 检索提示

- 12-24 个月，儿歌串烧，经典童谣，动作歌，重复节奏。
