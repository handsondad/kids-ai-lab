# 游戏简介

这是一个适合 3-8 个月宝宝的身体部位命名活动。家长边轻轻摸宝宝的脸、鼻子、肚子、腿和脚趾，边把这些部位的名字慢慢说出来，让触觉和词语一起进入。

## 适用场景

- 换衣、换尿布、洗澡前后或地垫时间里
- 家长想把日常触摸变成更清楚的语言输入时

## 准备材料

- 不需要额外材料

## 玩法步骤

1. 让宝宝舒服地躺着或靠在你怀里。
2. 先从脸开始，轻轻摸一下并说“脸”。
3. 再带到眼睛、鼻子、嘴巴、肚子、腿和脚趾。
4. 说到每个部位时，也可以让宝宝摸一摸你的对应部位。
5. 一轮只做几个部位就够了。

## 引导话术

- “这是你的脸。”
- “这是鼻子。”
- “小脚趾在这里。”

## 观察点

- 哪些部位最容易引起宝宝兴趣或笑声
- 重复几次后，听到词时会不会更专注或主动碰那里
- 宝宝更喜欢你摸她，还是把手放到你脸上

## 难度变体

- 简化版：每次只说脸、鼻子和脚趾三个部位
- 进阶版：把短儿歌或固定节奏加进命名里

## 安全提示

- 动作保持轻柔，避开眼球和敏感区域的直接按压
- 宝宝明显抗拒触碰时，先减量或换时间再玩
