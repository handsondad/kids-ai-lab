# 游戏简介

这是一个适合 6-24 个月孩子的洗澡语言活动。家长在洗澡时把热水、泡泡、鸭子、小脚趾和舀水动作都说出来、唱出来，让清洁流程同时成为触觉和语言输入时段。

## 适用场景

- 洗澡时孩子已经比较放松，愿意听你声音时
- 家长想让澡盆时间更有互动，但又不想增加复杂玩具时
- 需要把清洁过渡做得更顺一点时

## 准备材料

- 洗澡本来就会用到的水、毛巾、小杯子或小鸭子等安全玩具

## 玩法步骤

1. 洗澡开始时先只说最明显的感觉，比如“暖暖的水”。
2. 洗到不同部位时，说出身体部位和动作，比如“洗洗脚趾”。
3. 如果有泡泡、杯子或小鸭子，再把它们也说进来。
4. 可以把一句简单短话唱出来，让节奏更稳定。
5. 洗完时用固定结束语收尾，比如“泡泡洗好了，我们擦干啦”。

## 引导话术

- “暖暖的水来了。”
- “这里有泡泡。”
- “洗洗你的小脚趾。”
- “小鸭子在漂呀漂。”

## 观察点

- 孩子更喜欢听说话、听唱，还是看你操作小工具
- 哪些词最容易形成熟悉感：泡泡、水、脚趾、鸭子
- 洗澡前后语言输入会不会让过渡更顺一些

## 难度变体

- 极简版：只固定 2-3 个洗澡词
- 进阶版：加上冷热、漂、倒、满、空这类感觉和动作词
- 歌曲版：把固定洗澡句子变成一小段重复旋律

## 安全提示

- 洗澡和玩水都必须全程近距离看护，不能因为说话分神离开
- 以安全洗浴为先，语言输入是附加，不要让流程变乱

## 检索提示

- 6-24 个月，洗澡说话，泡泡语言，卫生间话语，日常对话输入。
