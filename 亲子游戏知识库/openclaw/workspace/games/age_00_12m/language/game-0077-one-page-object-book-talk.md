# 游戏简介

这是一个适合 8-18 个月孩子的低龄对话式共读活动。家长选图片简单、每页只有一两个熟悉物体的书，围绕颜色、质地、名称和动作去聊，而不是急着把整本书讲完。

## 适用场景

- 孩子已经愿意翻页、摸书页或盯着大图看几秒时
- 想把共读做得更有互动，而不是只朗读时
- 家长想练习“停下来聊一句”时

## 准备材料

- 一本图片清楚、物体少、最好有一点触摸或翻页互动的低龄书

## 玩法步骤

1. 让孩子能同时看到书页和你翻页的动作。
2. 打开一页后，先说出最明显的一个物体或颜色。
3. 停一下，等孩子看、摸、翻或发声。
4. 再顺着他的动作补一句，比如质感、颜色、会做什么。
5. 一次只聊几页，保持阅读和对话都短而清楚。

## 引导话术

- “这里有一只鸟。”
- “这页摸起来软软的。”
- “你想翻这一页吗？”
- “你刚刚一直在看这个杯子。”

## 观察点

- 孩子会不会因为你停下来而更愿意看、摸或发声
- 哪类页面最能让他参与：翻翻页、摸摸页，还是颜色鲜明的大图
- 你每次补多少信息最合适，不会太快把他淹没

## 难度变体

- 极简版：每页只说一个名称和一个颜色词
- 进阶版：在固定几页上加同一句重复话术，让孩子更容易预期
- 触摸版：优先选有质感或翻翻机关的书

## 安全提示

- 低龄孩子仍可能咬书和撕书，优先用结实安全的材质
- 共读目标是一起看和一起聊，不追求从头到尾完整读完
