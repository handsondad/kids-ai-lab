# 游戏简介

这是一个适合 3-8 个月宝宝的童谣互动活动。家长挑一两首简单、重复、朗朗上口的童谣，边唱边加一点手势、摇摆或指向动作，让宝宝同时接触旋律、节奏和语言模式。

## 适用场景

- 宝宝喜欢听你唱歌，也愿意看你做动作时
- 想把语言输入做得更有节奏、更容易重复时

## 准备材料

- 一两首你熟悉的短童谣

## 玩法步骤

1. 选一首最短、最容易重复的童谣。
2. 一边唱，一边做很简单的手势，比如壶把、倒水、蜘蛛爬、轻轻摇摆。
3. 每唱到关键词，就停半拍，让宝宝看你的动作和表情。
4. 一首唱两三遍就可以，不必不断换歌。
5. 最后用同一句收尾语结束，让宝宝知道这轮结束了。

## 引导话术

- “我们来唱这首小歌。”
- “小蜘蛛爬呀爬。”
- “茶壶倒出来啦。”

## 观察点

- 宝宝更容易被旋律、节奏还是动作吸引
- 哪一首歌最容易让他安静下来或露出笑
- 重复第二遍时，会不会出现明显期待感

## 难度变体

- 简化版：只唱一首，不加太多动作
- 进阶版：把宝宝的小手也轻轻带进一个简单动作里

## 安全提示

- 手势和摆动都保持小幅度，不做过强刺激
- 歌词不用一次很多，重点是重复和熟悉感

## 检索提示

- 3-8 个月，童谣，儿歌，节奏语言，音乐欣赏，动作歌。
