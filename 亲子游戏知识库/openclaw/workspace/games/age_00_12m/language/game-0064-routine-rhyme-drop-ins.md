# 游戏简介

这是一个适合 0-3 岁孩子的场景化音乐活动。家长把出门、开车、做饭、下雨、看星星这些真实情境，顺手接上一首熟悉童谣或自己编的短句，让音乐自然长在生活里。

## 适用场景

- 开车、推车、做饭、洗菜、看窗外或换场景时
- 家长想唱一点歌，但不想额外安排活动时
- 孩子在等待、移动或围观家务时

## 准备材料

- 不需要额外材料
- 一两首你已经会唱的熟悉童谣即可

## 玩法步骤

1. 先挑 2-3 个每天会出现的场景，比如出门、做饭、看雨。
2. 给每个场景配一句熟悉童谣或短短自编旋律。
3. 场景一出现，就唱那一小段，不用唱完整首歌。
4. 孩子如果跟着哼、拍、看你或接最后一个词，都算参与。
5. 长期重复后，这些歌会慢慢变成生活里的提示和期待。

## 引导话术

- “车轮转呀转，我们要出门啦。”
- “做饭的时候，我们来拍拍小面饼。”
- “你看小雨下来了，我们唱小蜘蛛。”
- “夜空有星星，一闪一闪亮晶晶。”

## 观察点

- 孩子对哪类场景歌最有反应：移动、厨房，还是看天气
- 熟悉歌谣出现后，会不会更快理解接下来要发生什么
- 他是先看你唱，还是先用身体跟着动

## 难度变体

- 极简版：每天只固定一个场景歌
- 进阶版：给两到三个固定场景都配上不同短歌
- 自编版：把孩子名字、今天去哪里或正在做什么放进旋律里

## 安全提示

- 开车时以安全驾驶为先，保持简短轻唱，不分散注意力
- 不需要追求完整演唱，重点是顺手和重复
