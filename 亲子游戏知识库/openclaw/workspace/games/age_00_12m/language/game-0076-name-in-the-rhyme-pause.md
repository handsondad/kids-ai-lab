# 游戏简介

这是一个适合 6-18 个月孩子的童谣互动活动。家长把孩子名字编进熟悉歌谣里，唱到关键位置时稍微停一下，等孩子用眼神、声音、笑或动作接上，让唱歌慢慢变成来回交流。

## 适用场景

- 孩子对熟悉童谣有明显反应时
- 想把唱歌从单向表演变成互动时
- 路上、换场景前或面对面坐着时都能做

## 准备材料

- 不需要额外材料
- 一首你最熟悉、重复句比较多的童谣

## 玩法步骤

1. 先把孩子名字替换进一首熟悉歌谣里。
2. 唱几遍完整版本，让孩子先熟悉节奏。
3. 到固定位置时轻轻停 1-2 秒，看孩子会不会发声、看你、动起来或笑。
4. 不管回应大小，都立刻接住，把歌继续唱完。
5. 重复几轮后结束，保持活动短而轻松。

## 引导话术

- “今天这首歌里有你的名字。”
- “唱到这里，轮到你啦。”
- “你刚刚接住了这句歌。”

## 观察点

- 孩子会不会在停顿前就开始预期那一拍
- 是名字出现更能吸引他，还是停顿结构更能吸引他
- 他更常用哪种方式接：发声、看你、拍手还是晃身体

## 难度变体

- 极简版：只在固定一个位置做停顿
- 进阶版：加入简单动作，如拍手或点头，再停顿等回应
- 录音版：把你们最熟悉的一小段录下来，过几天再一起听

## 安全提示

- 不把停顿做成考验，孩子没接上就自然唱下去
- 重点是高成功率的来回感，而不是接得准不准

## 检索提示

- 6-18 个月，名字编进歌，停顿接唱，童谣互动，来回交流。
