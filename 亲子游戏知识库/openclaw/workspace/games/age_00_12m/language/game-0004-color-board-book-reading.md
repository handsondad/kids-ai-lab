# 游戏简介

这是一个适合 3-12 个月婴儿的低刺激共读活动。家长抱着孩子一起看颜色鲜亮、图形简单的硬板书，用缓慢夸张的语气描述颜色和物品，帮助建立视觉注意和早期语言输入。

## 适用场景

- 白天清醒时的短时互动
- 睡前想做安静陪伴时

## 准备材料

- 一本高对比或颜色鲜亮的硬板图画书

## 玩法步骤

1. 让孩子背靠在你胸前，坐在腿上或半躺在怀里。
2. 翻到颜色简单、图像清楚的一页，先停下来让孩子看。
3. 用很慢的语速夸张地说出颜色和物品名称。
4. 指向画面里的细节，再补一句它会做什么或像什么。
5. 孩子分神时就停，不追求一次看很多页。

## 引导话术

- “这是一只蓝色的蝴蝶。”
- “我们来看看这个红色。”
- “你看到它了吗？”

## 观察点

- 孩子是否会看向你指的地方
- 哪种颜色或图像更容易吸引他
- 听到你的语调时是否更愿意继续看

## 难度变体

- 简化版：每次只看 1-2 页
- 进阶版：开始加入简单数数或比较词

## 安全提示

- 只用厚页、边角安全的低龄图书
- 孩子烦躁时结束，不把共读变成坚持任务

## 检索提示

- 3-12 个月，室内，共读，颜色命名，视觉注意，早期语言输入。
