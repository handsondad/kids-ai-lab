# 游戏简介

这是一个适合 6-12 个月宝宝的拉取再现活动。家长把一条柔软丝巾穿进纸筒或布筒，让一端露出来，宝宝一拉，整条丝巾就慢慢跑出来，形成很直观的“消失又出现”。

## 适用场景

- 宝宝已经很喜欢抓布、拉布和看东西被带出来时
- 想把精细动作和物体恒存性放进同一个活动里时

## 准备材料

- 一条轻薄、整洁、无脱线的丝巾或布条
- 一个纸筒或边缘安全的软筒

## 玩法步骤

1. 先把丝巾穿过筒，只露出一小截给宝宝看。
2. 示范轻轻拉一下，让更多布慢慢出现。
3. 把露出的那一端递给宝宝，让他自己继续拉。
4. 拉完后再慢慢塞回去，开始下一轮。
5. 你可以边拉边描述“出来了、还在里面、全出来啦”。

## 引导话术

- “丝巾藏在里面。”
- “拉一拉，它出来啦。”
- “还有一点点在里面。”

## 观察点

- 宝宝看到只露一点时，会不会立刻想继续拉
- 会不会用一只手拉、另一只手接住出来的布
- 拉的过程中是否明显更专注地等待结果

## 难度变体

- 简化版：多露出一些丝巾，降低起始难度
- 进阶版：换成两种颜色的布，比较先后出现的变化

## 安全提示

- 丝巾必须全程看护，不用于遮脸或独自玩耍
- 纸筒边缘要平整，无卷裂和硬刺

## 检索提示

- 6-12 个月，丝巾纸筒，拉出来，再出现，精细动作。
