# 游戏简介

这是一个适合 9-18 个月宝宝的掉落因果活动。家长坐在宝宝餐椅或地面旁边，给宝宝几个安全轻物，让她往下放、丢给你或掉到地上，再把东西递回去，体验“下去”和“回来”的循环。

## 适用场景

- 宝宝最近特别爱把东西从高处往下丢时
- 家长想把高频掉东西行为变成一个更可控的游戏时

## 准备材料

- 2-4 个轻巧、安全、不会碎的塑料杯、摇铃、积木或小毛绒玩具

## 玩法步骤

1. 把一两个物件放在宝宝面前的托盘或腿边。
2. 坐在旁边接住或捡起物件，不需要一次给太多。
3. 当宝宝把东西往下放或丢时，马上描述“下来了”。
4. 稍停一下，再把物件递回去开始下一轮。
5. 玩几分钟就收，避免无限循环变成大人过度疲劳。

## 引导话术

- “它掉下去啦。”
- “现在又回来了。”
- “你再试一次。”

## 观察点

- 宝宝是更喜欢看东西掉，还是更期待你把它递回来
- 哪种物件最容易引发重复尝试：会响的、会滚的，还是软的
- 你放慢一点递回去时，她会不会更专注地等结果

## 难度变体

- 简化版：由宝宝把物件递给你，不一定要丢下去
- 进阶版：换两种不同落地声音的物件，比较结果差异

## 安全提示

- 只用轻巧安全物件，不在玻璃、热饮或尖角附近玩
- 如果宝宝明显开始用力往人脸方向扔，就立刻收并改到更安全的放下玩法

## 检索提示

- 9-18 个月，扔东西，掉下去，上下因果，餐椅游戏，重复实验。
