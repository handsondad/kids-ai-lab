# 游戏简介

这是一个适合 7-11 个月宝宝的扶站起步活动。家长在地毯上让宝宝从仰躺到坐起，再在稳定扶持下轻轻拉到站立，同时把简单儿歌和左右手词汇说进去。

## 适用场景

- 宝宝已经愿意在大人扶持下站一小会儿时
- 想把起身动作和面对面互动结合起来时

## 准备材料

- 一块柔软、防滑的地毯或地垫
- 一首简单身体动作歌，可选

## 玩法步骤

1. 让宝宝仰躺在地垫上，你握住他的双手或前臂。
2. 先轻轻把他带到坐起位置，停一下。
3. 再慢慢带到站立，但始终稳稳扶住，不让他自己承重太久。
4. 站起后对视、说一句固定短话，再慢慢回到坐或躺。
5. 做两三轮即可，不追求次数。

## 引导话术

- “我们慢慢坐起来。”
- “现在站一站。”
- “右手、左手。”

## 观察点

- 宝宝从躺到坐、从坐到站，哪一步更需要帮助
- 站起来后是更开心，还是更需要立刻回到坐姿
- 固定歌谣和词汇会不会让动作更可预测

## 难度变体

- 简化版：只做到坐起，不拉到站立
- 进阶版：站起后加一个轻轻摇摆或拍手收尾

## 安全提示

- 全程由成人承担稳定扶持，不让宝宝突然向后倒
- 不拉手腕猛提，动作必须缓慢而连续

## 检索提示

- 7-11 个月，扶站起步，拉坐拉站，身体儿歌，面对面互动。
