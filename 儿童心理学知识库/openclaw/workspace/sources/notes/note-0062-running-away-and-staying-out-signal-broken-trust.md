# 核心观点

- 阿德勒把青春期离家出走、夜不归宿和反复逃离家庭，理解为对家庭束缚、过度保护和缺乏自由感的一种反抗，而不只是单纯的任性或寻欢。
- 即使青少年没有真的离家出走，只要他总在找机会夜不归宿、尽量少待在家里，也常能看见类似倾向。对孩子来说，这往往意味着家已经不再被体验成一个能信任、能自由呼吸、能被理解的地方。
- 阿德勒同时指出，友谊和可信任的成人关系是预防青春期问题的关键保护因素。只有当父母和教师能被青春期孩子视作朋友般可信赖的人，他们才可能继续发挥教育影响。
- 如果父母或教师做不到这一点，很快就会被青春期孩子视为外人甚至敌人。此时再继续用命令和控制去压，往往只会把关系推得更远。
- 对今天的转译来说，这部分最重要的提醒是：先理解青少年到底在逃离什么、在向谁发出控诉，再决定怎样处理夜不归宿和离家风险，而不是只看表面行为。

## 可拆出的阶段卡或支持卡

- 阶段卡：青春期会把童年的位置感、关系准备和证明冲动一起显影
- 支持卡：青春期想离家、夜不归宿或彻底不信任家里时，先修自由与信任，不只加码控制
