# 核心观点

- 阿德勒提醒，不要把“妹妹出生了，所以哥哥变坏了”理解成简单因果链。真正推动问题行为恶化的，不是事件本身，而是儿童对事件形成的错误认知和由此设定的目标。
- 个体心理学不强调机械因果，而强调在发展过程中不断出现的大大小小的错误判断。这些判断会干扰儿童原本的发展轨迹，让他沿着更狭窄、更失真的方向努力。
- 儿童很小就会为自己设定追求卓越的目标，并且用这个目标来组织接下来的行为。如果目标建立在错误判断上，后续行为即使很一致，也会把孩子带向越来越糟的方向。
- 同一个客观事实，对不同儿童会激起不同理解。第二个孩子出生这件事本身没有唯一的心理后果，关键在于孩子把它理解成什么，是“我被夺走了位置”，还是“家里多了一个人”。
- 决定行为的不是事物本身，而是个体对事物的看法。儿童遇到新困难时，往往仍会沿着自己已经形成的认知方式去反应，因此错误认知会在新情境中持续重演。

## 可拆出的阶段卡或支持卡

- 阶段卡：3-6 岁生活风格开始成形，同一目标会在不同行为里重复出现
- 支持卡：二孩出生后先处理被取代感和错误目标，不只惩罚攻击和退行

## 家长会怎么问

- 为什么家里只是多了一个孩子，大宝就像变了一个人
- 为什么同样是二孩家庭，有的孩子适应，有的孩子越来越糟
- 孩子老是重复同一种问题，是不是因为他已经形成固定想法了
