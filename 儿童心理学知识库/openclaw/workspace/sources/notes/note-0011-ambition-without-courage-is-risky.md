# 核心观点

- 阿德勒并不把强烈野心视为纯粹优点。野心如果脱离勇气、毅力和面对困难的能力，反而会让孩子更脆弱、更依赖外界认可。
- 这类孩子最在意的，不是自己是否真的成长了，而是别人有没有看见、赞赏、承认自己的成功。一旦没有被认可，再大的成绩也很难让他们满足。
- 过度野心会制造长期紧张感。孩子可能只盯成绩、盯排名、盯超越别人，忽视游戏、户外活动、同伴关系和身心平衡。
- 当这类孩子面对真实困难时，常常不是更能扛，而是更容易崩。因为他们把失败理解成价值坍塌，而不是一个可解决的问题。
- 阿德勒更看重的教育方向，不是把孩子变成野心勃勃的人，而是培养勇气、毅力、自信和面对失败时不绝望的能力。

## 可拆出的阶段卡或支持卡

- 阶段卡：4-6 岁过强的优越感追求会放大竞争、嫉妒和失败焦虑
- 支持卡：别把好胜心都当优点，要练面对失败的勇气

## 家长会怎么问

- 孩子成绩很好，但越来越紧张和不快乐，要不要担心
- 为什么越优秀的孩子，有时越怕失败、越怕别人失望
- 激将法、低分刺激和比较竞争，到底有没有用
