# 核心观点

- 对自卑、气馁、误入歧途的孩子来说，最需要的不是威胁、嘲讽和“你以后一定完了”这种预判，而是勇气、信心和可以渐进练出来的独立感。
- 阿德勒明确反对用羞辱来激孩子改。孩子因为怕被嘲笑而做出的改变往往只是表面动作，内里并没有真正建立面对事实和面对困难的能力。
- 一个已经自卑或怯懦的孩子，在尊严受损时，很可能会做出更错误、更危险的补偿性举动，例如为了掩饰“我不会”而做出鲁莽冒险行为。
- 因此，真正有效的教育，不是抓住孩子的软肋去刺，而是通过友好关系、持续鼓励、让孩子逐步凭自己的力量完成事情，来恢复他对自身能力的信心。
- 预判孩子的坏未来，本质上也是一种伤害。它会进一步拿走孩子对未来的希望，让其更相信“我反正不行”。

## 可拆出的阶段卡或支持卡

- 阶段卡：3-6 岁自卑情结会让孩子在退缩、敌意和特殊症状之间摇摆
- 支持卡：嘲讽和预判坏未来只会加深自卑，先恢复勇气和独立感
- 支持卡：口吃和表达困难里，先保护勇气和关系，不要紧盯每个音

## 家长会怎么问

- 骂醒孩子、激将孩子，到底为什么经常没用
- 为什么孩子越被笑，就越做出更离谱的事情
- 怎样鼓励孩子，才不会变成空泛安慰
