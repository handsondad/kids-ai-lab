# 核心观点

- 阿德勒在这一节说得很直接：任何会激怒父母、令其感觉被羞辱或被逼到墙角的教育方式，都几乎不可能换来真正有用的合作。
- 尤其是在一些贫困家庭里，父母本就更习惯用责罚和打骂来回应问题。如果学校的反馈方式又进一步点燃父母的羞耻和怒气，孩子回家后就很可能为同一件事再挨一轮更重的处罚。
- 这样一来，学校的努力就会被家庭里的不合理教育方式抵消，甚至放大成更严重的后果。孩子接收到的不是帮助，而是“同一个错误会让我在两个地方都付出代价”。
- 因此，教育者必须把“父母回家后可能怎么做”算进自己的方法里，而不能只看自己在学校这一端是否讲清楚、罚到位或表达了立场。
- 这一部分最值得今天保留的结论是：如果一种教育做法会稳定地把家庭推向更粗暴、更羞辱的方向，那么它即使当下看起来严厉有效，整体上也仍然是在伤害孩子。

## 可拆出的阶段卡或支持卡

- 阶段卡：4-6 岁孩子在家校之间的适应问题，常取决于成人能否协作、不双罚并整体看人
- 支持卡：学校已经反馈问题、回家还要再罚时，先停双重惩罚，别把孩子逼向逃学、藏匿和伪造

## 家长会怎么问

- 为什么有些老师反馈完问题，孩子回家反而更糟
- 学校和家里都在罚，为什么事情还是没变好
- 为什么激怒家长的做法，最后会转头伤到孩子
