# 核心观点

- 案例四里的 8 岁半男孩每年留级、言谈像婴儿、不喜欢上学、和哥哥打架、怕黑、极度挑食。阿德勒认为，这些并不是零散毛病，而是被长期溺爱后的典型后果：孩子学会了像更小、更需要被照顾的人那样占据母亲的注意力。
- 晚走路、母亲长时间围着照料、父亲也慈爱纵容，再加上和哥哥的竞争，都让他更相信“保持更小、更需要被宠爱”对自己更有利。
- 因此学校对他构成真正的测试，因为学校要求合作、独立和现实任务，而他并没有接受过这种训练，于是只好用婴儿化、扰乱、留级和对抗来应付。
- 这个案例非常重要的一点在于，阿德勒把学校视为预防性场域：如果此时还不帮助孩子恢复独立与勇气，后面更可能沿着逃学、停学、离家和混入无益团体的方向走。
- 这一案例最值得今天保留的结论是：被溺爱的学校期儿童，最需要的不是更强压力，而是重新学会独立、允许犯错、扩大有用活动范围，并在学校里遇到能鼓舞勇气的大人。

## 可拆出的阶段卡或支持卡

- 阶段卡：6-12 岁进入学校比较与群体秩序后，问题行为常会沿着争中心、避失败和躲合作三条线长出来
- 支持卡：孩子总像更小、不能独处或一离开围着转就乱时，先补独立训练和有用角色，不只加学校压力
- 支持卡：忘带书本、拖延和逃避上学时，先修复学业信心
