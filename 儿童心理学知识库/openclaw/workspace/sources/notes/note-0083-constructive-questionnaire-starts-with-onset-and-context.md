# 核心观点

- 附录Ⅰ一开始就提醒，分析问题儿童时，不应先问“这孩子到底坏在哪里”，而应先问“问题是从什么时候开始的”，以及“当时这个孩子把自己放在什么处境里”。
- 阿德勒列出的重要线索不是抽象标签，而是转折点，例如环境改变、开始上学、弟妹出生、换教师、换学校、新朋友、生病、父母离婚、再婚或死亡。这些事件本身不是自动原因，但常常是旧脚本最先显影或被放大的节点。
- 这也意味着，问卷的真正功能不是收集零碎缺点，而是帮助我们找到问题行为第一次变得必要的那一刻：孩子当时是在争位置、逃难题、保自尊、求关注，还是在对某种新处境做出误读。
- 对今天的转译来说，这一部分最值得保留的结论是：任何分析都应该从“起点-处境-变化”开始，而不是从“毛病目录”开始。
- 这也是为什么附录最后特别强调，问答不应按死板顺序进行，而应以建设性谈话的方式展开；因为真正要找的是孩子如何解释自己的处境，而不是怎样把他逐条归类。
