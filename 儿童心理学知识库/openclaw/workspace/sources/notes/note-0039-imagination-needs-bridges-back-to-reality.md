# 核心观点

- 阿德勒并不把白日梦和想象力本身视为罪过。幻想有时只是孩子在绕开现实痛苦、保护自己不被失败再次刺伤时走的一条弯路。
- 一部分孩子会在幻想和现实之间找到折中路径，一边继续保留想象，一边慢慢适应现实；一部分孩子会越来越从现实抽离；还有一部分孩子会把幻想转移到真实世界中的英雄故事、旅行见闻、历史典故和可观察的现实素材上。
- 关键不在于彻底消灭幻想，而在于帮助孩子理解现实生活的价值和意义，让幻想与现实之间不至于裂开成不可逾越的鸿沟。
- 阿德勒同时承认，丰富的想象力如果能和勇气、观察力、面对现实的能力结合，反而是领导力、创造力和成就的重要土壤。问题只在于，孩子有没有勇气把幻想中的力量一点点放回真实生活。
- 因此，成人不能粗暴、鲁莽地对待这类孩子。真正有效的是鼓励、解释现实的意义、帮助其从小处连接现实，而不是用羞辱把他进一步赶回幻想壳里。

## 可拆出的阶段卡或支持卡

- 阶段卡：3-6 岁孩子会按家庭位置写出价值脚本，并在乖、闹、退和幻想之间选路
- 支持卡：孩子总躲进白日梦、显得冷淡摆烂时，先修勇气和现实连接，不急着把他当没追求
- 支持卡：孩子总是非赢即输、非黑即白时，先松动对立思维，不和他硬碰硬
- 支持卡：别把懒惰只当不上进，先看它是不是在保护脆弱信心

## 家长会怎么问

- 孩子爱幻想，到底是问题还是资源
- 怎么让沉迷幻想的孩子重新接回现实生活
- 为什么粗暴打压想象力，反而会让孩子更退
