# 核心观点

- 那些看起来懒惰、内向、沉默、对规则和批评都无动于衷的孩子，并不等于没有追求优越感。阿德勒认为，他们只是不相信自己能靠常规方式成功，于是退回到幻想与抽离之中。
- 这类孩子表面冷淡麻木，内里却往往非常敏感、焦虑、容易受伤。他们像是给自己套上一层盔甲，用“我不在乎”“我不参与”“你碰不到我”来保护脆弱心灵。
- 白日梦是这种保护的一种典型形式。孩子会在幻想里把自己设想成英雄、统治者、拯救者或成就非凡的人物，以此补偿现实中的无力感和低位置感。
- 重要的是，这类幻想不必然意味着彻底放弃。只要勇气还没有完全崩掉，这些孩子在现实中也可能真的去训练自己、模仿英雄、等待能证明价值的机会。
- 因此，面对沉迷幻想和抽离现实的孩子，重点不是嘲笑其“不务正业”，而是理解：他正在用幻想撑住一个在现实里很难承受失败的自己。

## 可拆出的阶段卡或支持卡

- 阶段卡：3-6 岁孩子会按家庭位置写出价值脚本，并在乖、闹、退和幻想之间选路
- 支持卡：孩子总躲进白日梦、显得冷淡摆烂时，先修勇气和现实连接，不急着把他当没追求
- 支持卡：别把懒惰只当不上进，先看它是不是在保护脆弱信心

## 家长会怎么问

- 为什么有的孩子总活在幻想里，不愿意面对现实任务
- 孩子看起来冷冷的、懒懒的，是不是根本没上进心
- 为什么有些孩子嘴上说无所谓，脑子里却全是当英雄的幻想
