# 核心观点

- 阿德勒指出，许多教师都觉得与问题儿童父母沟通，比直接与孩子工作还难。这不是因为父母天然难缠，而是因为家长一被叫到学校，很容易立刻感到自己像被传唤、被控诉、被要求为孩子的问题负责。
- 在这种情境下，家长心里常同时带着羞愧、防御和紧张。如果教师继续使用指责、定责或揭短的方式，家长会更快进入对抗，而不是进入合作。
- 因此教师更合适的位置，不是法官，也不是控诉者，而是父母的助手和协作者。重点不是证明谁错了，而是争取父母的善意配合，一起改变孩子的行为路线。
- 阿德勒甚至主张，在与父母沟通时，教师应预设“父母不必为孩子的所有问题负责”。因为很多父母并不是专业教育者，他们只是沿用了传统、过时却熟悉的方法。
- 这一部分最值得今天保留的结论是：家校会谈能不能帮到孩子，第一步往往不在于分析多深，而在于有没有先把会谈从“追责现场”改成“共同帮孩子找路”。

## 可拆出的阶段卡或支持卡

- 阶段卡：4-6 岁孩子在家校之间的适应问题，常取决于成人能否协作、不双罚并整体看人
- 支持卡：一聊孩子问题就容易互相防御时，先去罪责化，争取善意配合，不把会面开成控诉
- 支持卡：学校和家庭都觉得卡住时，先共看问题链条，不互相甩锅

## 家长会怎么问

- 老师一找家长谈话，为什么家长常常先防御起来
- 家校沟通怎样才不会变成追责
- 为什么说老师更像助手，而不是法官
