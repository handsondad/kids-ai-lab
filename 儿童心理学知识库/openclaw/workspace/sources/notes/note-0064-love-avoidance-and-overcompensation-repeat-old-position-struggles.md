# 核心观点

- 阿德勒认为，青春期青少年在爱情、亲密和与异性交往上的态度，并不会凭空变出一套全新人格，而往往是在更强烈的情境里重复之前的生活风格。
- 那些表面上对亲密很抗拒、对异性很消极、很怕柔软和依赖的孩子，常常不是天生冷淡，而是把更早以前的争位置、失位、嫉妒和骄傲保护带进了新的关系任务里。
- 也就是说，青春期亲密困难不只是在谈“恋爱技巧”，更是在谈一个人如何处理自己被爱、被看见、被比较、被拒绝和不必时刻逞强这件事。
- 阿德勒还指出，职业方向、人际困难和爱情态度常常彼此相连，共同显出一个人面对成年生活准备是否充分。
- 因此，青春期里对亲密和爱情的极端态度，不宜只被当作道德问题或流行问题，更适合被理解为旧有关系脚本在新场景中的重复。

## 可拆出的阶段卡或支持卡

- 阶段卡：青春期会把童年的位置感、关系准备和证明冲动一起显影
- 支持卡：青春期把恋爱、性和离家当成争关注或反击时，先追旧失位和被欣赏饥饿，不只做道德审判

## 家长会怎么问

- 为什么孩子一到青春期就对亲密特别抗拒或特别逞强
- 孩子对异性态度极端，是不是和童年经历有关
- 爱情问题为什么常和学业、工作、人际困难绑在一起
