# 核心观点

- 阿德勒并不主张因为课程有压力就简单减少学科数量；更重要的是，用系统、具体、可感的方式去教，让孩子知道所学内容的目的和现实价值。
- 在他看来，学科知识和人格培养并不冲突。真正好的教学，是把知识、兴趣、现实用途和合作能力一起长出来，而不是让课程只剩抽象理论和被动记忆。
- 这也是他赞同综合教学、散步观察、学科互联的原因。教学越能与真实世界相连，孩子越容易形成贯通性的思维，而不仅是为了应付测试。
- 班级理想上应该像一个整体，每个孩子都能在其中感到自己属于这个整体。教师需要有意识地限制过度竞争，把孩子的精力从“我怎么压过别人”转向“我们怎么一起做成事情”。
- 阿德勒还提醒，很多孩子会低估自己，并把几次失败理解成永久位置。教师必须及早纠正这种“我永远不如别人”的认知，否则它会稳定成一辈子的自我设限。

## 可拆出的阶段卡或支持卡

- 阶段卡：4-6 岁入学后，兴趣、归属和希望比分数更早决定学校走向
- 支持卡：数学、游泳或上学任务一再卡住时，先修复安全感，不用排名放大无能感
- 支持卡：先找到孩子已经能投入的学科和感官通道，用强项带动弱项

## 家长会怎么问

- 为什么孩子觉得课程枯燥时，怎么逼都没用
- 班级竞争为什么会把有些孩子越推越差
- 孩子总觉得自己永远比不上别人，老师和家长该怎么接
