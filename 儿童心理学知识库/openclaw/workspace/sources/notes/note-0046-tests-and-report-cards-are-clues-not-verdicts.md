# 核心观点

- 阿德勒并不否认智力测验和学业评价的参考价值，但他强烈反对把这些结果理解成对儿童能力和命运的最终裁决。
- 智力测验能提供普通考试看不到的信息，某些时候甚至能帮助孩子摆脱不合理的留级决定；但一旦家长和孩子把“分数”听成“你就是这样的人”，测验就会反过来限制发展。
- 阿德勒强调，测验成绩并不是固定不变的实质本体。只要找到正确的训练和准备方法，孩子完全可能在后续测验中提高表现，这恰恰说明它更像当下状态的取样，而不是永久标签。
- 对成绩单也应如此理解。成绩单不只是记录答对多少题，还会在某种程度上反映孩子当下的兴趣、专注、准备度和心智运作状态。
- 因此，无论是智力测验还是成绩单，最有价值的用法都不是“定义孩子”，而是“帮助我们更准确地理解孩子此刻卡在哪里”。

## 可拆出的阶段卡或支持卡

- 阶段卡：4-6 岁入学后，兴趣、归属和希望比分数更早决定学校走向
- 支持卡：成绩单、智力测验和留级建议都只是线索，不是命运判决
- 支持卡：忘带书本、拖延和逃避上学时，先修复学业信心

## 家长会怎么问

- 智力测验到底能不能说明孩子聪不聪明
- 成绩单到底应该怎么看，才能不把孩子吓坏
- 为什么有些低分孩子后来也能翻上来
