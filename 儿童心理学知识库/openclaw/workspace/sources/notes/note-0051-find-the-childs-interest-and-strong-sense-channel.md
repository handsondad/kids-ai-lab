# 核心观点

- 在阿德勒看来，真正有效的教育往往不是先盯住最差的地方，而是先找到孩子已经愿意投入、能够做成、能从中获得成就感的地方，再把这份成功感迁移到其他领域。
- 所谓“一事成功，事事顺利”，不是神奇口号，而是说：孩子在一个学科或任务里尝到真实成功后，更容易长出继续学习其他内容的勇气。
- 他还强调，教师要去识别孩子更擅长使用哪类感官通道来学习和理解世界。例如有的孩子更偏视觉，有的更吃听觉或动作体验；当教学方式贴近孩子更强的输入方式时，学习就更容易被接住。
- 这类优势不应只被当成“偏科”或“特殊爱好”，而应被当作通向更广泛学习和自我效能感的桥梁。
- 因此，教育者真正要做的，不是反复告诉孩子“你哪里差”，而是先找到“你已经能靠什么站住”，再让这块可站住的地方慢慢扩大。

## 可拆出的阶段卡或支持卡

- 阶段卡：4-6 岁入学后，兴趣、归属和希望比分数更早决定学校走向
- 支持卡：数学、游泳或上学任务一再卡住时，先修复安全感，不用排名放大无能感
- 支持卡：先找到孩子已经能投入的学科和感官通道，用强项带动弱项

## 家长会怎么问

- 孩子只有一门课愿意学，这能不能成为突破口
- 怎么判断孩子更吃图像、声音还是动手体验
- 为什么先从强项入手，比一直补短更有效
