# 核心观点

- 阿德勒认为，社会情感发展程度可以通过孩子的典型行为来判断：如果一个孩子在追求优越时总只盯着自己、不考虑别人，那么其社会情感通常发展不足。
- 但这里最重要的不是道德审判，而是理解根源。长篇道德说教改变不了真正失去方向的孩子，尤其当孩子已经开始走向严重问题行为时，说教只会让他更快相信自己“就是坏、就是笨、就是不行”。
- 更可取的立场不是法官，而是伙伴或医生。教育者需要追问行为背后的环境作用，例如：一个不会整理的孩子身边常有替他整理的大人；一个撒谎成性的孩子身边常有专制强硬、急于严厉纠正的大人；一个爱吹嘘的孩子常常更在意被夸，而不是把事情真正做好。
- 这意味着，孩子身上的“错”往往也记录着环境的误配。环境越不利、越误解、越压低孩子的信心，孩子越可能用错误行为去证明那个错误判断是真的。
- 因此，看见问题行为时，教育者要从“这孩子品行差”转向“环境如何持续地把他推回这个位置”。

## 可拆出的阶段卡或支持卡

- 阶段卡：3-6 岁社会情感发展不足时，孩子更容易退回自我中心、表达退缩和争位置
- 支持卡：别凭一个行为给孩子下结论，用关系-任务-亲密三问看孩子
- 支持卡：孩子撒谎、吹嘘、不会整理或总想赢时，先停下道德审判，回到处境和环境

## 家长会怎么问

- 孩子总撒谎、吹牛、不收拾，是不是品行有问题
- 为什么越讲道理、越批评，孩子越改不好
- 看见孩子自私和爱争，是不是只能靠讲道德去压
