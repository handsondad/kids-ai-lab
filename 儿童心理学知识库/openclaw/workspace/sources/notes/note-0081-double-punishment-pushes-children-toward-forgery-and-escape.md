# 核心观点

- 阿德勒在这里给出一个非常具体的链条：如果孩子拿着糟糕成绩单回家需要签名，而他已经知道回家会挨打挨骂，那么他面对的不是简单“承担后果”，而是两边夹击。
- 这时孩子很容易被逼到几种边缘路线里：逃学、旷课、不回家，或者干脆伪造父母签名。问题并不是他天生更坏了，而是双重惩罚把他推向了更极端的保身办法。
- 这一分析的价值很大，因为它提醒我们：一些看上去更严重的行为，并不总是原发问题，有时只是成人设计的后果结构把孩子逼到了那里。
- 阿德勒因此要求教育者不断自问：如果我坚持这样做，接下来会发生什么？孩子能承受吗？他会从中获得什么经验？
- 这部分最值得今天保留的结论是：任何后果设计，只要稳定把孩子逼向遮掩、逃跑和伪造，就说明它已经偏离教育，而更像在制造新问题。

## 可拆出的阶段卡或支持卡

- 阶段卡：4-6 岁孩子在家校之间的适应问题，常取决于成人能否协作、不双罚并整体看人
- 支持卡：学校已经反馈问题、回家还要再罚时，先停双重惩罚，别把孩子逼向逃学、藏匿和伪造
- 支持卡：孩子一被罚就更硬、更乱时，先停下惩罚循环，回到错误目标和适应缺口

## 家长会怎么问

- 为什么成绩单签名这种事会逼出更大的问题
- 孩子伪造签名、逃学，是不是就说明更坏了
- 怎么判断一种后果是不是已经把孩子逼偏了
