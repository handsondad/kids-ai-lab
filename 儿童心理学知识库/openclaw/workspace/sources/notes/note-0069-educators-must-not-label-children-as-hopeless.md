# 核心观点

- 阿德勒开篇先指出，教育者最大的失误之一，不是暂时没有看到成果，而是在没有立刻成功时就先丧失勇气和信心，把孩子定义成懒散、消极、无药可救或没有天赋。
- 对他来说，教育的任务不是提前给孩子判能力死刑，而是努力保护勇气、激发心智发展，并让孩子逐步明白困难不是不可逾越的天命，而是可以被面对和练习的任务。
- 这也意味着，教育成效不能只按短期回报来判断。有些孩子改变得快，有些慢，但只要成人不急于把孩子钉死在“就是不行”的身份上，就仍有机会打开新的路线。
- 阿德勒并不否认现实中有些努力收效有限，但他更强调，真正拖垮孩子的常常不是困难本身，而是成人先相信“这个孩子已经没希望了”。
- 因此，这一节最值得今天保留的结论是：教育中的首要失误，往往是成人先撤走了希望，然后再把孩子的退缩和摆烂当作证据。

## 可拆出的阶段卡或支持卡

- 阶段卡：12-18 岁长期被贴成不行、迟钝或没希望时，孩子会更容易退回更小、更安全的位置
- 支持卡：被“没天赋、没救了、就是差”围住时，先拆失败身份，别让孩子继续照着判决活
- 支持卡：青春期一乱就先当旧脚本显影，不把变化全算成突然叛逆
