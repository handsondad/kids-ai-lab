# 核心观点

- 阿德勒把学业表现，尤其是数学能力，看作一个重要的心理指标，因为数学、写作、绘画、体操、舞蹈、游泳等活动，都和“我能否在混乱中获得稳定感、掌控感和安全感”有关。
- 某些孩子在大部分科目都不错，却在数学上持续挫败；这种挫败会不断确认其“我不够行”的感觉。于是问题往往不只是学科难度本身，而是学科把孩子的不安全感放大了。
- 社会上关于性别和学科能力的刻板结论，也会提前拿走孩子的勇气，让孩子在真正尝试前就先选择退开。
- 学不会游泳、在身体技能上持续胆怯，也常常提示孩子对自己和对带领者缺少信任；但一旦能跨过困难，很多这类孩子反而会因为对困难高度敏感而发展出很强的精进动力。
- 这部分提醒教育者，面对学业或技能困难时，不宜只用成绩和比较来施压，而要看孩子有没有安全感、是否还相信自己可以学会。

## 可拆出的阶段卡或支持卡

- 阶段卡：3-6 岁预防自卑情结，关键是理解处境、保住勇气和独立性
- 支持卡：忘带书本、拖延和逃避上学时，先修复学业信心
- 支持卡：数学、游泳或上学任务一再卡住时，先修复安全感，不用排名放大无能感

## 家长会怎么问

- 孩子为什么一到数学、游泳或某类任务就特别怂
- 为什么越比较成绩，孩子越不想学
- 孩子的学习问题到底是不会，还是已经不敢
