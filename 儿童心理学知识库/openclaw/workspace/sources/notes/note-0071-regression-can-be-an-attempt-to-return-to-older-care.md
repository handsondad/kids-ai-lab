# 核心观点

- 阿德勒对男孩把自己年龄说成 11 岁这一细节非常敏感，因为在他看来，这并不是随口说错，而是在暴露一种更深的退行愿望：想回到更小、更脆弱、也更容易得到照顾和宽宥的时候。
- 同样的路线也表现在白天尿裤、无法控制排便，以及整个人在某些时刻更像一个更低龄孩子。这些并不只是“坏习惯”或“故意淘气”，而是孩子正在用行为把自己重新放回一个更小的位置上。
- 为什么要这么做？因为在那个更小的阶段，他更容易获得专门照顾，也不必面对与同龄人竞争、被评价能力、被看作失败者的痛感。
- 因此，退行在这里不是随机散乱的，而是沿着同一条路线在工作：如果我不能通过胜任和成长得到位置，那我就退回到需要被照顾的样子，通过脆弱来获得安全。
- 对今天的转译来说，这一部分最值得保留的结论是：大孩子的退行表现，常常不是单点症状，而是一种“回到更小才更安全”的整体策略。

## 可拆出的阶段卡或支持卡

- 阶段卡：12-18 岁长期被贴成不行、迟钝或没希望时，孩子会更容易退回更小、更安全的位置
- 支持卡：大孩子说小自己年龄、反复退行或偷小东西争看护时，先看他是不是在退回更小更安全的位置

## 家长会怎么问

- 为什么一个 12 岁孩子还会像更小的小孩那样退回去
- 说错年龄、尿裤子这类细节为什么要重视
- 退行是不是在告诉我们他不想长大
