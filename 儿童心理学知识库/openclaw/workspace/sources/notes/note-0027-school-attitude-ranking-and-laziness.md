# 核心观点

- 孩子对学校的态度，是预防自卑情结的重要观察点。迟到、紧张、作业前暴躁、反复忘作业、厌学和拖延，往往都不只是纪律问题，也可能是在逃离会再次证明自己无能的地方。
- 阿德勒反对以频繁考试、排名和成绩分类的方式组织学校生活，因为这种制度容易把一时成绩变成像终身判决一样的心理负担。
- 懒惰在这一部分被说得更清楚：有些孩子宁可被看成懒，也不愿被看作无能；只要还可以说“我没认真”，就能暂时逃开“我真的不行”的打击。
- 对教师、自习和作业的态度也很关键。有些孩子通过捣乱、羞辱同学、自大聒噪来遮住弱点；有些孩子则戴上冷漠、消极、什么都不做的面具，以避免被失败刺痛。
- 最需要的不是简单命令，而是鼓励、支持与更现实的小成功经验。那些在体育、阅读或某项特定领域特别用力的孩子，也常常是在其他受挫领域之外寻找补偿性的优越感。

## 可拆出的阶段卡或支持卡

- 阶段卡：3-6 岁预防自卑情结，关键是理解处境、保住勇气和独立性
- 支持卡：忘带书本、拖延和逃避上学时，先修复学业信心
- 支持卡：别把懒惰只当不上进，先看它是不是在保护脆弱信心
- 支持卡：数学、游泳或上学任务一再卡住时，先修复安全感，不用排名放大无能感

## 家长会怎么问

- 为什么孩子越怕学校，越拖越忘越发脾气
- 孩子说不想学，到底是懒还是怕被证明不行
- 排名、比较和公开评判为什么会让孩子更退
