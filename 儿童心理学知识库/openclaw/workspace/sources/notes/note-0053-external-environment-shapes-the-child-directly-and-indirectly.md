# 核心观点

- 阿德勒强调，儿童的心理从来不是只靠内在冲动自行发展出来的，外部环境会像持续不断的波浪一样涌入儿童心灵，对其产生直接或间接的影响。
- 所谓间接影响，常常是环境先改变父母的心态、生活方式和情绪基调，再由父母把这种心态传给孩子。因此，研究儿童不能只研究孩子，也不能只研究环境，而要看环境对孩子和照养者各自意味着什么。
- 这也是阿德勒认为个体心理学不需要再分出一门“社会心理学”的原因之一。对他来说，个体从来就是在社会关系和环境处境中被理解的。
- 对教育者和教师而言，这一章最重要的提醒是：大人不是儿童的唯一塑造者。家庭经济、病痛经历、家庭气候、亲戚朋友、读物玩具、偏见和社会风气，都会一起进入教育现场。
- 因此，真正的教育工作不能只盯孩子眼前的一个表现，而要持续追问：最近有哪些外部力量正在重塑这个孩子对世界、对他人、对自己的理解。

## 可拆出的阶段卡或支持卡

- 阶段卡：3-6 岁时，贫困、病痛、家庭气候与媒介脚本会一起塑造勇气和社会兴趣
- 合集：第十一章关于外部环境、病痛、偏见、玩具和祖辈影响的专题合集

## 家长会怎么问

- 为什么明明我在教育孩子，最后影响他的却不止我一个人
- 外部环境到底会怎样进入孩子心里
- 看孩子问题时，为什么总要连带看大人的处境和社会环境
