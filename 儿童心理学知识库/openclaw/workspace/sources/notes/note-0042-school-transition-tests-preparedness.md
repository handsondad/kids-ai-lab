# 核心观点

- 阿德勒认为，新环境之所以重要，不是因为它制造了问题，而是因为它清楚地测试了孩子是否已经准备好。准备足的孩子会更有信心地进入新环境；准备不足的孩子会被无能感扭曲判断，做出与环境需求不符的反应。
- 学校尤其如此。孩子在上学前一切似乎正常，上学后却明显崩掉，这往往是在提示“学校这个新环境把原本隐藏的缺口显影了”，而不是学校单独把孩子弄坏了。
- 因而，观察问题何时出现、环境在那之前发生了什么变化、问题在变化前是否早有迹象，都是理解适应不良的关键线索。
- 阿德勒还提醒，很多早期信号都很容易被忽视，例如孩子做事没条理、总有人替他收拾，孩子特别胆小退缩、过度依恋家庭，或在身体笨拙、发育迟缓、长期被过度照顾或被忽视中积累了不安。
- 面对这些孩子，教师和家长的任务不是用体罚、辱骂和“你没出息”坐实他们的无能感，而是培养勇气、重新建立现实信心，并通过更细致的观察看见行为背后的准备缺口。

## 可拆出的阶段卡或支持卡

- 阶段卡：3-6 岁遇到新环境时，旧脚本会被显影，准备度和归属感决定适应质量
- 支持卡：孩子一到新环境就出问题时，先把它当准备度测试，不把适应失手当故意
- 支持卡：入园入学先看适应与归属，不只盯表现
- 支持卡：孩子一被罚就更硬、更乱时，先停下惩罚循环，回到错误目标和适应缺口

## 家长会怎么问

- 为什么孩子一上学才真正暴露问题
- 怎么判断是新环境太难，还是孩子准备不足
- 孩子成绩差、适应差，被打被骂为什么往往更糟
