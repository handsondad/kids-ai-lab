# 核心观点

- 附录Ⅰ后面几项问题专门追问：孩子在哪些方面失去信心，是否觉得自己被忽视，是否相信自己缺乏天赋，是否因周围人的态度越来越悲观，是否靠装傻、做鬼脸、故作幼稚来吸引注意，是否反复谈论自己的无能，甚至是否出现自杀想法。
- 这说明阿德勒并不是只关心症状目录，而是非常关心孩子的勇气到底丢在了哪里，以及孩子后来发明了哪些替代路线来弥补这种丧气。
- 同时，他还要求分析者列出孩子获得成功的事例。这个要求非常关键，因为它意味着评估不该只围绕失败和毛病，也要回收孩子真正有兴趣、有准备、曾经成功过的方向。
- 最后，阿德勒明确说，完成这些问答时不应采用例行公事或固定顺序，而应以建设性谈话方式进行，并始终用宽容、耐心、友好的态度解释孩子的问题，避免指责、威胁和逼迫。
- 对今天的转译来说，这一部分最值得保留的结论是：好的评估既要找到孩子怎样失去勇气，也要找到他还在哪些地方保有力量；而整个过程必须像帮助，不像审讯。

## 家长会怎么问

- 为什么评估孩子问题时，还要特意列出成功经验
- 分析孩子时，怎样做到不带威胁和指责
- 总说自己没天赋、没希望，为什么在问卷里这么重要
