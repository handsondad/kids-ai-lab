# 核心观点

- 阿德勒提醒我们，在同一个家庭里，一个孩子表现“很好”，另一个孩子表现“很糟”，并不等于前者天生更善良、后者天生更差。两者都可能是在争取价值感和重要性，只是路线不同。
- “好孩子”可能已经意识到，自己只要继续表现得乖、优秀、让人满意，就能稳定赢得父母认可；而“坏孩子”也在追求优越，只是他发现自己在“做好”这条赛道上难以胜出，于是转向相反方向，通过顽劣、吵闹、对抗来获取注意和支配力。
- 这意味着，淘气顽劣本身也可能是强烈优越感追求的结果。一个孩子之所以能被推向极端顽劣，也说明他本来就拥有很强的驱动力，只是这股力量被导错了方向。
- 阿德勒举出的案例还说明，所谓“好”与“坏”都可能随着家庭情境变化而翻转。孩子今天靠乖顺获得价值，明天也可能在位置变化、欲望受阻时改用哭闹、借钱、对抗来继续满足旧目标。
- 所以，教育者不能只看当前表现，而要看孩子在用什么手段追求被看见、被肯定和被满足，以及当旧手段失效时，他会转向哪条新路线。

## 可拆出的阶段卡或支持卡

- 阶段卡：3-6 岁孩子会按家庭位置写出价值脚本，并在乖、闹、退和幻想之间选路
- 支持卡：家里一个乖一个闹时，先看他们各自争取价值和关注的路线，不急着固定好坏孩子

## 家长会怎么问

- 为什么一个孩子越教越省心，另一个越教越闹
- 孩子明明以前很乖，为什么后来突然变得难搞
- 顽劣是不是就说明孩子更坏、更自私
