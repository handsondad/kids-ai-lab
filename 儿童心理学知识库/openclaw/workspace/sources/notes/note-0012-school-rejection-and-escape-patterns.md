# 核心观点

- 个体心理学视角下，所谓“差生”常常不是不想追求优越，而是把追求优越的努力导向了对学校的拒绝、逃避或搞破坏。
- 当孩子在学校里一再体验到自己不行、不会成功、得不到正向位置时，他可能会转向更容易获得注意、崇拜或存在感的路径，例如恶作剧、挑衅、逃学、与不良同伴结伙。
- 这类行为的内核不是单纯懒散，而是学业自信已经崩塌，孩子开始逃避那些会持续暴露无能感的任务。
- 学校本该是家庭与社会之间的桥梁，也有责任帮助纠正家庭阶段形成的错误生活方式。但如果学校只用抽象成绩来定义孩子，就容易进一步强化失败感。
- 教育者更有效的做法，是从孩子已表现出来的某个能力或成就出发，帮助他把“我在这方面能行”的体验迁移到其他任务，而不是只盯住缺点反复打压。

## 可拆出的阶段卡或支持卡

- 阶段卡：4-6 岁过强的优越感追求会放大竞争、嫉妒和失败焦虑
- 支持卡：忘带书本、拖延和逃避上学时，先修复学业信心

## 家长会怎么问

- 孩子是不是单纯懒，还是已经对学校失去信心了
- 为什么有些孩子宁可捣乱，也不愿正经做任务
- 如果孩子在某个领域能做好，能不能借这个把学业信心拉起来
