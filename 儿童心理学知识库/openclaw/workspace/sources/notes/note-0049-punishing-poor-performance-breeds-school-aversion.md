# 核心观点

- 阿德勒反复强调，糟糕成绩本身已经足够让孩子承受羞耻与挫败；如果学校和家庭再叠加批评、责骂、打骂和羞辱，孩子自然会越来越讨厌学校，并想尽办法逃开这个地方。
- 对很多孩子来说，差成绩单不是“提醒我要努力”，而是“我又要回家受罚”的信号。于是恐惧、隐瞒、伪造成绩单、逃学甚至更严重的结伴失控行为，就会沿着这条线长出来。
- 阿德勒特别反对留级被当成默认处理方式。因为留级往往不是在解决问题，而是在公开确认“你就是落后的人”，很容易把孩子的无望感钉得更死。
- 同样，简单把贫困家庭、入学准备不足的孩子放进慢班，也会迅速把“准备不足”变成“你就是差生”的标签，伤害其信心与归属感。
- 因此，真正要防的不是一张难看的成绩单，而是孩子从这张成绩单里听出了“我没希望了，我必须离开这个地方”。

## 可拆出的阶段卡或支持卡

- 阶段卡：4-6 岁入学后，兴趣、归属和希望比分数更早决定学校走向
- 支持卡：忘带书本、拖延和逃避上学时，先修复学业信心
- 支持卡：成绩单、智力测验和留级建议都只是线索，不是命运判决
- 支持卡：学校和家庭都觉得卡住时，先共看问题链条，不互相甩锅
