# 核心观点

- 阿德勒把“能力来自遗传、差就是天生差”看成教育中最危险的误区之一。因为它会同时给父母、教师和孩子提供一个最省事的借口，让所有人都逃开本来应该承担的支持责任。
- 他并不否认生理器官和身体条件可能存在差异，也承认心理会感受到这些差异并做出反应；但这不等于能力和成就被遗传宿命锁死。
- 在他的视角里，很多所谓“家族遗传的才能”更像家风、期待、机会结构和训练方式的传承，而不是直接的能力基因继承。
- 因此，如果把学校里的失败轻易归结为遗传，就会忽略孩子当下真正缺的可能是准备、训练、鼓励、方法、归属感或更合适的支持环境。
- 对教育工作来说，最重要的不是追问孩子为什么不像某个天才祖先，而是追问现在怎样做，才能让他把原本可能长出来的能力长出来。

## 可拆出的阶段卡或支持卡

- 阶段卡：4-6 岁入学后，兴趣、归属和希望比分数更早决定学校走向
- 支持卡：成绩单、智力测验和留级建议都只是线索，不是命运判决

## 家长会怎么问

- 孩子成绩差，是不是就说明脑子不行、遗传不好
- 为什么总把问题怪到遗传上，会妨碍教育
- 身体条件差一点，会不会注定学不好
