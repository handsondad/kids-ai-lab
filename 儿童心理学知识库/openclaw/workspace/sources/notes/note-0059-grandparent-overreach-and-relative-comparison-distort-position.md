# 核心观点

- 阿德勒对祖父母和外祖父母持一种复杂但同情的理解：他们之所以频繁干预教养，常常并不只是因为爱孩子，而是也在通过“我还很有用、我还懂教育”来证明自己仍有价值。
- 问题在于，这种证明自己价值的方式很容易落到过度溺爱、过度插手、让孩子站队、或者用孩子来调节成人冲突上。这样一来，孩子就会被卷进不属于他的关系和权力问题里。
- 被祖辈偏爱的孩子，往往更容易遇到纵容、竞争和他人嫉妒等复杂后果；而一旦发现自己并不是“最受宠的那个”，也会强烈受伤。
- 阿德勒还提醒，那些才华出众、外貌出众的堂表兄弟姐妹，也常成为孩子的重要外部压力源。如果成人总拿这些同辈作参照，孩子很容易把差距听成“别人天生比我强”，从而落入自卑、怨恨或放弃。
- 因此，真正要守住的不是禁止祖辈接近孩子，而是避免祖辈用孩子证明自己，也避免成人用优秀同辈、美貌或天赋不断制造高低比较。

## 可拆出的阶段卡或支持卡

- 阶段卡：3-6 岁时，贫困、病痛、家庭气候与媒介脚本会一起塑造勇气和社会兴趣
- 支持卡：祖辈太介入、总拿优秀同辈比较时，先守住教养主轴，不让宠溺和攀比占位
- 支持卡：祖辈宠溺、母子过密和反复制造麻烦时，先修复独立性，不让照顾变控制

## 家长会怎么问

- 为什么祖辈越爱孩子，有时越容易把孩子带偏
- 总拿聪明漂亮的亲戚孩子比较，会留下什么后果
- 祖辈和父母意见不一时，怎样不让孩子被夹在中间
