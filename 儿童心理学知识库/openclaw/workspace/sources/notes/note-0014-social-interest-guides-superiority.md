# 核心观点

- 阿德勒认为，正确引导孩子追求优越感的关键标准不是“赢得多不多”，而是这种努力是否走向了对社会有益、对关系有益、对问题解决有益的方向。
- 他把这个判断标准称为社会兴趣。一个孩子若只想证明自己高人一等，却不愿意把力量放进合作、贡献和现实解决问题中，这种追求很容易走偏。
- 行为是否有价值，不能只看当下是否得利，还要看持续效果和最终结果。与现实规律、合作关系和长期成长相契合的行为，才更可能是有建设性的方向。
- 孩子偏离对社会有益的道路时，常常不能自动从负面后果中吸取积极教训，因为他们并没有理解整条生命主线和自己选择之间的关系。
- 教育者需要帮助孩子看到：生命不是一个个彼此无关的事件，而是一条连续的路径。孩子若能理解自己如何一步步走向某种结果，才更有可能修正方向。

## 可拆出的阶段卡或支持卡

- 阶段卡：4-6 岁正确引导优越感，关键是把自我证明转向社会兴趣和解决问题
- 支持卡：别把懒惰只当不上进，先看它是不是在保护脆弱信心

## 家长会怎么问

- 怎么判断孩子的上进心是在成长，还是已经走偏了
- 为什么有些孩子越想证明自己，反而越容易出问题
- 教孩子追求优秀，到底该以什么标准来校正方向
