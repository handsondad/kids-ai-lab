# 核心观点

- 阿德勒把“新环境”看成观察人格的显影液。人格具有连续性，新的情境不会凭空制造一个全新的人，却会把平日被熟悉环境包住的部分冲洗出来。
- 因此，孩子开始上学、进入新家庭、照养关系突变、家庭遭遇重大变化时，往往是最适合理解其生活方式的时刻。此时孩子隐藏的恐惧、目标和旧解释会更清楚地露出来。
- 关键不在成人自认为给了多少爱或多好条件，而在孩子怎样解释自己的处境。被领养的孩子即使客观上被照顾得不错，也可能仍把新家庭体验成牢笼，或坚信自己不是被同样爱着的人。
- 阿德勒用被收养男孩的案例说明，很多看似暴躁、危险、答非所问的行为，若从孩子自己设定的目标来看，其实都服务于一个一致的方向，例如“我要离开这个家”。
- 这类孩子的问题，不是单靠外在“对他更好一点”就能扭转的，因为真正决定行为的，是他心里对归属、偏爱、禁锢和自由的解释。

## 可拆出的阶段卡或支持卡

- 阶段卡：3-6 岁遇到新环境时，旧脚本会被显影，准备度和归属感决定适应质量
- 支持卡：孩子一到新环境就出问题时，先把它当准备度测试，不把适应失手当故意
- 支持卡：被领养、继亲或总觉得自己不是自己人时，先修归属感，不让管教被听成排斥

## 家长会怎么问

- 为什么孩子一换环境就像变了一个人
- 明明已经很爱他了，为什么孩子还是一直对抗
- 孩子进入新家庭或新集体后很敌对，是不是故意不知好歹
